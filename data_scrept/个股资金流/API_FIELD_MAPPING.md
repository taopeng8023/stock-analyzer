## 📋 API 接口信息

**接口地址**: `https://push2.eastmoney.com/api/qt/clist/get`

**请求参数**:
```
pn=1              # 页码
pz=100            # 每页数量
po=1              # 排序方向
np=1              # 
ut=bd1d9ddb04089700cf9c27f6f7426281
fltt=2            # 
invt=2            # 
wbp2u=|0|0|0|web
fid=f62           # 排序字段（根据排行类型变化）
fs=m:0+t:6,m:0+t:80,m:1+t:2  # 市场筛选（沪深 A 股）
fields=f12,f14,f2,f3,...     # 返回字段
```

## 💾 数据格式化规则

### 金额字段
- ≥ 1 亿：显示为 `X.XX 亿`
- ≥ 1 万：显示为 `X.XX 万`
- < 1 万：显示为 `X.XX`

### 百分比字段
- 统一显示为 `X.XX%`

## 🔧 使用示例

### Python 脚本

```python
# 导入字段映射
FIELD_MAPPING = {
    "f12": "代码",
    "f14": "名称",
    "f62": "今日主力净流入 - 净额",
    # ... 更多字段
}

# API 请求
import requests
url = "https://push2.eastmoney.com/api/qt/clist/get"
params = {
    "pn": 1,
    "pz": 100,
    "fid": "f62",  # 今日排行
    "fields": "f12,f14,f2,f3,f62,f184,f66,f67,f68,f69,f70,f71,f72,f73"
}
response = requests.get(url, params=params)
data = response.json()

# 字段映射
for stock in data['data']['diff']:
    item = {}
    for api_field, value in stock.items():
        if api_field in FIELD_MAPPING:
            chinese_name = FIELD_MAPPING[api_field]
            item[chinese_name] = format_value(value, chinese_name)
```
