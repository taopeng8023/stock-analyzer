## 💡 实战策略

### 策略 1: 主力大幅流入股

**条件**:
- 主力净流入 > 1 亿
- 主力占比 > 5%
- 综合评分 ≥ 70
- 成功概率 ≥ 60%

**操作**: 重仓买入 (30-50%)

### 策略 2: 板块轮动

**步骤**:
1. 获取行业板块排名
2. 找出连续 3 日净流入的板块
3. 选择板块龙头股
4. 分析个股基本面

**操作**: 中线持有 (1-3 个月)

### 策略 3: 板块联动

**逻辑**:
- 板块大涨 → 找补涨股
- 板块大跌 → 规避风险
- 连续流入 → 中线布局

**操作**: 短线快进快出 (5-10 天)

### 策略 4: 综合选股

**流程**:
```
1. 板块排名 → 找出热门行业
2. 主力排名 → 找出资金流入股
3. 选股系统 → 综合评分筛选
4. 基本面分析 → 排除问题股
5. 技术面确认 → 选择买点
```

## 📈 评分系统详解

### 综合评分计算

```python
total_score = (
    技术面 * 0.35 +    # K 线、均线、趋势
    基本面 * 0.35 +    # PE、PB、ROE
    资金面 * 0.30      # 主力流入、占比
)
```

### 买入建议阈值

| 综合评分 | 成功概率 | 建议 | 仓位 |
|---------|---------|------|------|
| ≥80 | ≥70% | 强烈推荐买入 | 30-50% |
| ≥70 | ≥60% | 推荐买入 | 25-35% |
| ≥60 | ≥50% | 观望 | 10-20% |
| <60 | <50% | 不建议买入 | 0% |

### 资金面评分标准

| 主力净流入 | 评分加成 | 主力占比 | 评分加成 |
|-----------|---------|---------|---------|
| > 1 亿 | +20 | > 10% | +15 |
| > 0.5 亿 | +15 | > 5% | +10 |
| > 0 | +5 | > 0% | +5 |
| < -1 亿 | -20 | < 0% | 0 |
| < -0.5 亿 | -15 | - | - |

## 🎯 板块分析应用

### 行业板块

**用途**: 发现行业轮动机会

**用法**:
- 连续流入 → 中线布局
- 单日爆发 → 短线跟进
- 大幅流出 → 规避风险

### 概念板块

**用途**: 追踪市场热点

**用法**:
- 新概念 → 第一时间参与
- 老概念 → 注意持续性
- 概念轮动 → 提前布局

### 地区板块

**用途**: 区域经济分析

**用法**:
- 政策利好地区 → 重点关注
- 经济发达地区 → 稳健选择
- 欠发达地区 → 题材炒作
