### ⚠️ 重要修正：RSI计算方法 (2026-04-11 20:00)

**发现：我的RSI计算方法与同花顺不一致**

| 方法 | 皇台酒业RSI | 说明 |
|------|------------|------|
| 简单平均(旧) | 38.89 | ❌ 我的旧方法 |
| **EMA/Wilders** | **41.23** | ✅ 同花顺标准 |

**修正要点：**
- 同花顺使用EMA/Wilders平滑法计算RSI
- 已修正 `calc_rsi()` 函数使用Wilders平滑
- 算法：初始平均 + 后续平滑递推

**核心代码：**
```
# Wilders平滑法(14周期)
for d in deltas[period:]:
    if d > 0:
        gains = (gains*(13) + d) / 14
        losses = losses*(13) / 14
    else:
        gains = gains*(13) / 14  
        losses = (losses*(13) + (-d)) / 14
rsi = 100 - 100/(1 + gains/losses)
```

**教训：**
- 技术指标计算必须与主流软件一致
- 发现差异立即用实盘数据验证
- 主人截图纠错非常重要
