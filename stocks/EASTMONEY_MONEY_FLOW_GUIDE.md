## 🎯 数据源说明

### 东方财富资金流向 API

**接口地址**: `http://push2.eastmoney.com/api/qt/clist/get`

**参数说明**:
```
pn: 页码
pz: 每页数量 (最大 500)
po: 排序 (1=降序)
fid: 字段 ID
  - f4001 = 主力净流入
  - f4002 = 主力净流入占比
  - f4003 = 超大单净流入
  - f4004 = 大单净流入
fs: 市场筛选
  - m:0+t:6 = 沪市 A 股
  - m:0+t:80 = 深市 A 股
  - m:1+t:2 = 创业板
  - m:1+t:23 = 科创板
fields: 返回字段
  - f12 = 代码
  - f14 = 名称
  - f2 = 现价
  - f3 = 涨跌幅
  - f4001 = 主力净流入
  - f4002 = 主力净流入占比
```

## 📈 主力数据应用策略

### 策略 1: 主力持续流入

**条件**:
- 主力净流入 > 1 亿
- 主力占比 > 5%
- 连续 3 日净流入

**操作**: 跟踪买入

### 策略 2: 主力大幅流入

**条件**:
- 主力净流入排名 TOP 10
- 主力占比 > 10%
- 股价涨幅 < 5%

**操作**: 及时跟进

### 策略 3: 主力背离

**条件**:
- 股价下跌
- 主力大幅净流入
- 主力占比 > 8%

**操作**: 左侧布局机会

## ⚠️ 注意事项

### 1. 数据时效性
- 盘中数据每 3 分钟更新
- 收盘后数据固定
- 建议 14:30 后使用

### 2. 数据准确性
- 主力定义：超大单 + 大单
- 超大单：单笔成交≥100 万或≥50 万股
- 大单：单笔成交≥20 万或≥10 万股

### 3. 配合其他指标
- ✅ 结合技术面分析
- ✅ 结合基本面分析
- ✅ 结合市场情绪

### 4. 风险提示
- 主力数据可能被操纵
- 对倒交易干扰判断
- 需要综合判断
