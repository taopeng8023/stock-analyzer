## 🕐 交易时间判断

### 判断逻辑

```python
def is_trading_time():
    # 周末不交易
    if weekday >= 5:
        return False
    
    # 上午：9:30-11:30
    if hour == 9 and minute >= 30:
        return True
    if hour == 10:
        return True
    if hour == 11 and minute <= 30:
        return True
    
    # 下午：13:00-15:00
    if hour == 13:
        return True
    if hour == 14:
        return True
    if hour == 15 and minute <= 0:
        return True
    
    return False
```

### 特殊情况

| 情况 | 处理方式 |
|------|----------|
| **法定节假日** | cron 自动跳过（1-5 表示工作日） |
| **临时休市** | 推送空数据或提示 |
| **盘中停牌** | 显示停牌前数据 |
