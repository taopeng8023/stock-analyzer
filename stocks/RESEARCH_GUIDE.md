## 快速开始

### 1. 初始化数据库

```bash
cd /home/admin/.openclaw/workspace/stocks

# 初始化 SQLite 数据库
python3 research_db.py --init
```

### 2. 导入数据

#### 方法 A：生成示例数据（推荐新手）

```bash
# 生成 250 天示例数据
python3 research_import.py --sample --days 250
```

**说明：**
- ✅ 快速上手，无需外部数据源
- ✅ 学习数据库操作和数据分析
- ⚠️ 数据为随机生成，非真实市场数据

#### 方法 B：从 Tushare 导入真实数据

```bash
# 需要先配置 Tushare Token
python3 tushare_flow.py --config <your_token>

# 导入单只股票
python3 research_import.py --code 600000.SH

# 导入多只股票
python3 research_import.py --all
```

**说明：**
- ✅ 真实市场数据
- ✅ 可用于严肃研究
- ⚠️ 需要 Tushare 积分（120 积分解锁资金流）

#### 方法 C：从 CSV 导入

```bash
# 从已有 CSV 文件导入
python3 research_import.py --csv data.csv --code 600000.SH
```

### 3. 查看数据库状态

```bash
python3 research_db.py --stats
```

输出示例：

### 数据源对比

| 数据源 | 优点 | 缺点 | 适用场景 |
