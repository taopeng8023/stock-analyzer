## 🚀 快速使用

### 1. 完整工作流 (推荐)

```bash
cd /home/admin/.openclaw/workspace/stocks

# 运行工作流 + 推送 + 自动记录决策
python3 run_workflow.py --strategy all --top 10 --push --record
```

### 2. 回溯跟踪

```bash
# 跟踪进行中的决策
python3 run_workflow.py --backtest
```

### 3. 回测报告

```bash
# 生成汇总报告
python3 run_workflow.py --report

# 指定决策报告
python3 run_workflow.py --report --decision-id decision_20260320
```

### 4. 自动优化

```bash
# 基于历史表现优化权重
python3 run_workflow.py --optimize
```

## 📋 评级规则

| 条件 | 评级 | 置信度 |
|------|------|--------|
| 命中≥3 策略 且 涨幅>5% | 强烈推荐 | 90% |
| 命中≥2 策略 且 涨幅>3% | 推荐 | 75% |
| 命中≥2 策略 | 谨慎推荐 | 60% |
| 涨幅>5% | 关注 | 50% |
| 其他 | 观望 | 40% |
