## 🔧 数据源优先级

### 历史行情数据

```
优先级：缓存 (5 分钟) > 新浪 > 东财 (AKShare) > 腾讯
```

**原因**:
- **缓存**: 减少重复请求，最快速度
- **新浪**: 稳定，不限流，数据全
- **东财**: 限流严重，作为备用
- **腾讯**: 数据格式复杂，作为最后备用

### 资金流数据

```
优先级：东财 (AKShare) > 同花顺
```

**原因**:
- **东财**: 数据最全，包含超大单/大单/中单/小单
- **同花顺**: 备用数据源

### 热度数据

```
优先级：雪球 (AKShare) > 东财人气榜
```

**原因**:
- **雪球**: 真实用户关注数据，质量高
- **东财**: 人气榜，作为备用

## ⚠️ 限流问题说明

### 东方财富接口限流

**现象**:
```
❌ 获取失败：('Connection aborted.', RemoteDisconnected(...))
```

**原因**:
- 东方财富反爬严格
- 单 IP 请求频率限制
- 需要 Cookie/Token 认证

**解决方案**:
1. 使用新浪财经作为主数据源
2. 东财接口作为备用
3. 增加请求间隔 (限频保护)
4. 使用缓存减少重复请求

### 雪球接口限流

**现象**:
```
❌ 获取失败：需要 Token 认证
```

**解决方案**:
1. 配置雪球 Token (可选)
2. 使用关注榜接口 (无需 Token)
3. 热度数据作为辅助评分
