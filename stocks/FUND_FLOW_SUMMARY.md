## 🚀 推荐方案

### 方案 1：Tushare Pro（强烈推荐）

**优点：**
- ✅ 真实主力资金数据（特大单 + 大单）
- ✅ 免费开放（注册即送 100 积分）
- ✅ 数据稳定可靠
- ✅ 提供历史数据

**缺点：**
- ⚠️ 需要注册获取 Token（1 分钟完成）

**配置方法：**
```bash
# 1. 访问 https://tushare.pro 注册
# 2. 获取 Token
# 3. 配置
python3 tushare_flow.py --config <your_token>

# 4. 验证
python3 tushare_flow.py --check
```

**使用：**
```bash
# 主力净流入排行
python3 tushare_flow.py --top 20

# 集成到 fund_flow（自动优先使用）
python3 fund_flow.py --top 20
```

---

### 方案 2：腾讯财经估算（备选）

**优点：**
- ✅ 无需注册，开箱即用
- ✅ 数据稳定
- ✅ 覆盖全部 A 股

**缺点：**
- ⚠️ 估算数据（成交额×系数）
- ⚠️ 准确性一般

**算法：**
```python
# 根据成交量动态调整系数
if volume > 500 万手：factor = 0.18  # 18%
elif volume > 100 万手：factor = 0.15  # 15%
else: factor = 0.12  # 12%

主力净流入 = 成交额 × factor
```

**使用：**
```bash
python3 fund_flow.py --top 20 --source tencent
```

---
