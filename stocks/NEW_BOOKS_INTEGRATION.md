### 1. 艾略特波浪理论 (`elliott_wave.py`)

**核心功能**:
```python
# 波浪形态识别
wave_data = analyzer.identify_wave_pattern(kline_data)

# 输出:
{
    'pattern': '上升趋势',
    'wave_stage': '驱动浪 (可能第 3 浪)',
    'confidence': 0.75,
    'target': 8.50,  # 目标位
    'support': 7.00,  # 支撑位
}

# 交易信号
signal = analyzer.get_trading_signal(wave_data, current_price)

# 输出:
{
    'signal': 'buy',
    'reason': '驱动浪中，趋势强劲',
    'target': 8.50,
    'stop_loss': 6.80,
    'confidence': 0.75,
}
```

**评分集成**:
```python
# 波浪理论评分 (0-10 分)
score = elliott_wave_score(wave_data)

# 评分标准:
# - 驱动浪 + 高置信度：8-10 分
# - 调整浪 + 中置信度：5-7 分
# - 形态不明确：3-5 分
# - 下跌驱动浪：0-3 分
```

### 2. 江恩理论 (`gann_theory.py`)

**核心功能**:
```python
# 江恩综合分析
gann_data = analyzer.analyze(kline_data)

# 输出:
{
    'trend': 'bullish',
    'support_levels': [6.80, 7.00, 7.20],  # 支撑位
    'resistance_levels': [7.80, 8.00, 8.20],  # 阻力位
    'time_cycle': '接近 90 天周期 (±3 天)',
    'angle_position': {
        'position': '45 度线上方 (强势)',
        'signal': 'bullish',
    },
    'score': 7.5,  # 江恩评分
}

# 交易信号
signal = analyzer.get_trading_signal(gann_data)

# 输出:
{
    'signal': 'buy',
    'reason': '江恩评分 7.5，趋势向上，角度线强势',
    'target': 7.80,
    'stop_loss': 6.80,
    'confidence': 0.75,
}
```

**江恩关键要素**:
| 要素 | 说明 | 应用 |
|------|------|------|
| **角度线** | 45 度线最重要 | 判断趋势强弱 |
| **百分比** | 1/8, 2/8...8/8 | 支撑/阻力位 |
| **时间周期** | 30/45/60/90 天 | 变盘时点 |
| **价格轮动** | 1-8 轮动 | 目标位计算 |

**评分集成**:
```python
# 江恩理论评分 (0-10 分)
score = gann_score(gann_data)

# 直接返回 gann_data['score']
```

### 3. 资金管理 (`position_management.py`)

**核心功能**:
```python
# 仓位计算
manager = PositionManager(total_capital=100000)

position = manager.calculate_position(
    stock_data={'volatility': 0.03},
    entry_price=50.0,
    stop_loss=47.5,
    target_price=60.0
)

# 输出:
{
    'shares': 800,  # 建议股数
    'position_size': 40000,  # 仓位金额
    'position_pct': 40.0,  # 仓位占比%
    'risk_amount': 2000,  # 风险金额
    'risk_pct': 2.0,  # 风险占比%
    'reward_ratio': 4.0,  # 风险回报比
    'recommendation': '✅ 建议买入',
}

# 加仓/减仓策略
pyramid = manager.calculate_pyramiding(
    current_position={'entry_price': 50, 'shares': 800},
    current_price=55,
    new_entry=True
)

# 输出:
{
    'action': 'add',
    'shares': 400,  # 加仓 400 股
    'new_stop_loss': 52.50,
    'reason': '盈利 10.0%, 金字塔加仓',
}

# 移动止损
trailing_stop = manager.calculate_trailing_stop(
    entry_price=50,
    current_price=60,
    high_since_entry=62,
    method='gann'  # 江恩方法
)

# 输出: 58.25 (江恩 3/8 回撤位)
```

**斯波朗迪核心原则**:
| 原则 | 标准 | 应用 |
|------|------|------|
| **风险/回报** | 至少 1:3 | 过滤低质量交易 |
| **单笔风险** | ≤2% | 仓位计算基础 |
| **单股仓位** | ≤20% | 分散风险 |
| **盈利加仓** | 10% 以上 | 金字塔加仓 |
| **亏损减仓** | 5% 以上 | 止损减仓 |

**评分集成**:
```python
# 资金管理评分 (0-10 分)
score = position_management_score(position_data)

# 评分标准:
# - 风险回报比≥5: +3 分
# - 风险回报比≥3: +2 分
# - 仓位合理 (风险≤2%, 仓位≤20%): +2 分
```

### 综合评分计算

```python
def 综合评分 (股票数据):
    总分 = 0
    
    # 技术面 (30 分)
    总分 += 均线评分 * 0.4
    总分 += RSI 评分 * 0.3
    总分 += 量价评分 * 0.3
    
    # 波浪理论 (10 分) ✅ 新增
    波浪数据 = 波浪分析 (股票数据)
    总分 += elliott_wave_score(波浪数据)
    
    # 江恩理论 (10 分) ✅ 新增
    江恩数据 = 江恩分析 (股票数据)
    总分 += gann_score(江恩数据)
    
    # 策略面 (20 分)
    总分 += 回测评分 * 0.5
    总分 += 信号共振 * 0.5
    
    # 基本面 (15 分)
    总分 += 估值评分 * 0.5
    总分 += 行业评分 * 0.5
    
    # 资金管理 (10 分) ✅ 新增
    仓位数据 = 仓位计算 (股票数据)
    总分 += position_management_score(仓位数据)
    
    # 量价/CAN SLIM(5 分)
    总分 += 量价评分 * 0.5
    总分 += CANSLIM 评分 * 0.5
    
    return 总分

# 决策
if 总分 >= 80:
    建议 = "强烈推荐" ⭐⭐⭐⭐⭐
elif 总分 >= 60:
    建议 = "推荐" ⭐⭐⭐⭐
elif 总分 >= 40:
    建议 = "观望" ⭐⭐⭐
else:
    建议 = "回避" ❌
```
