## 🎯 热度指标对选股的作用

### ✅ 积极作用

| 作用 | 说明 | 适用场景 |
|------|------|----------|
| **发现热门股** | 快速找到市场关注焦点 | 短线选股 |
| **情绪指标** | 热度过高=顶部信号，过低=底部信号 | 反向操作 |
| **资金风向** | 热度上升通常伴随资金流入 | 趋势跟踪 |
| **流动性参考** | 热度高=流动性好=易进出 | 大资金选股 |
| **发现异动** | 热度突然上升=可能有消息 | 事件驱动 |

### ⚠️ 局限性和风险

| 风险 | 说明 | 应对方法 |
|------|------|----------|
| **滞后性** | 热度起来时股价已涨 | 结合技术面 |
| **反向指标** | 过热可能是出货信号 | 设置热度阈值 |
| **噪音大** | 热度不等于基本面 | 结合基本面分析 |
| **可操纵** | 热度可以人为制造 | 多数据源对比 |
| **时效短** | 热度变化快 | 实时跟踪 |

## 🚀 使用方法

### 1. 获取热度数据

```python
import akshare as ak

# 雪球关注榜
hot_data = ak.stock_hot_follow_xq(symbol='最热门')

# 东方财富热度上升榜
up_data = ak.stock_hot_up_em()
```

### 2. 热度评分

```python
def heat_score(rank, change=0):
    score = 0
    
    # 排名评分
    if rank < 10:
        score += 1  # 极热 (警惕)
    elif rank < 50:
        score += 2  # 高度关注
    elif rank < 200:
        score += 3  # 适度关注 (最佳)
    else:
        score += 2  # 关注度一般
    
    # 变化评分
    if change > 100:
        score += 3  # 热度飙升
    elif change > 50:
        score += 2  # 热度上升
    elif change > 0:
        score += 1  # 热度稳定
    
    return min(score, 5)
```

### 3. 选股系统调用

```python
# 获取热度排名
hot_data = ak.stock_hot_follow_xq()

# 查找股票排名
for i, row in hot_data.iterrows():
    if row['股票代码'] == symbol:
        rank = i + 1
        break

# 评分
score = heat_score(rank)
```
