**优化依据：** 2025-03 ~ 2026-03 历史回测结果  
**回测样本：** 50 只股票 × 12 个月 × 10 只/次 = 120 次决策

## 🔍 回测发现的关键问题

### 1. 评级表现倒挂
| 评级 | 决策次数 | 胜率 | 平均收益 |
|------|---------|------|---------|
| 强烈推荐 | 22 次 | 45.5% | -0.74% ❌ |
| 推荐 | 40 次 | 50.0% | -0.01% |
| 谨慎推荐 | 36 次 | 58.3% | +1.79% |
| 观望 | 22 次 | 59.1% | +3.82% ✅ |

**问题：** 激进评级表现差，保守评级表现好

### 2. 个股持续表现差
以下股票胜率<20% 且平均收益<-5%：
- 601138 工业富联 (0% 胜率，-13.53%)
- 301205 联特科技 (0% 胜率，-10.78%)
- 002460 赣锋锂业 (0% 胜率，-8.97%)
- 688629 华丰科技 (16.7% 胜率，-7.62%)

### 3. 止盈止损策略
- 止盈触发 44 次，止损触发 35 次
- 止盈/止损比 1.26:1，表现合理
- 但可优化：放宽止盈让利润奔跑，收紧止损快速止损

## ✅ v5.1 优化内容

### 1. 评级阈值优化

**强烈推荐 (大幅提高标准):**
```python
# 优化前
appear_count >= 2 and change_pct > 3%

# 优化后
appear_count >= 3 and change_pct > 5% and (signal == 'strong_buy' or signal == 'buy')
```

**推荐 (适度提高标准):**
```python
# 优化前
appear_count >= 2 and change_pct > 0%

# 优化后
appear_count >= 2 and change_pct > 3%
```

### 2. 黑名单机制

**黑名单股票 (直接降为观望，置信度 30%):**
- 601138 工业富联
- 301205 联特科技
- 002460 赣锋锂业
- 688629 华丰科技

**观察名单 (降一级处理，置信度 -15%):**
- 002475 立讯精密
- 300751 迈为股份
- 300342 天银机电
- 603026 石大胜华
- 000862 银星能源
- 300040 九洲集团

### 3. 评分权重调整

| 因子 | 优化前 | 优化后 | 变化 |
|------|-------|-------|------|
| 多日资金流 | 1.0× | 1.35× | ↑ 提高 |
| 主力排名 | 1.0× | 1.20× | ↑ 提高 |
| 基本面 | 1.0× | 1.15× | ↑ 提高 |
| 机构研报 | 1.0× | 1.10× | ↑ 提高 |
| 技术面 | 1.0× | 0.90× | ↓ 降低 |
| 量价分析 | 1.0× | 0.90× | ↓ 降低 |
| 趋势分析 | 1.0× | 0.90× | ↓ 降低 |

### 4. 止盈止损优化

| 涨跌幅 | 优化前 | 优化后 |
|-------|-------|-------|
| >10% | 止盈 +5% / 止损 -5% | 止盈 +8% / 止损 -6% |
| >5% | 止盈 +8% / 止损 -7% | 止盈 +12% / 止损 -8% |
| >0% | 止盈+10% / 止损-10% | 止盈+15% / 止损-10% |
| <0% | 止盈+12% / 止损-12% | 止盈+18% / 止损-12% |

**改进：** 止盈空间扩大 60-80%，让利润奔跑；止损保持或收紧

### 5. 置信度动态调整

**加分项:**
- 主力排名 Top20: +8%
- 主力排名 Top50: +5%
- 连续 5 日净流入：+8%
- 连续 3 日净流入：+5%
- 多策略共振 (4+): +8%
- 多策略共振 (3+): +5%
- 强烈买入信号：+10%

**减分项:**
- 黑名单股票：-30% (直接降为观望)
- 观察名单：-15% (降一级)

### 6. 评级分布控制

```python
RATING_DISTRIBUTION_LIMITS = {
    'strong_buy_max_pct': 0.15,  # 强烈推荐不超过 15%
    'buy_max_pct': 0.30,         # 推荐不超过 30%
    'cautious_buy_max_pct': 0.40, # 谨慎推荐不超过 40%
}
```
