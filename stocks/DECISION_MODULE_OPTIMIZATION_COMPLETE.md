## 🚀 使用方法

### 1. 单独使用模块

```python
from decision import (
    RiskControlModule,
    EnhancedTagGenerator,
    SmartStopLossCalculator,
    AdaptiveWeightManager,
    MultiDimensionScoring
)

# 风险控制
risk = RiskControlModule()
result = risk.check(stock_data)

# 标签生成
gen = EnhancedTagGenerator()
tags = gen.generate(stock_data, fundamental, technical)

# 止盈止损
calc = SmartStopLossCalculator()
stop_result = calc.calculate(stock_data, atr=50)

# 权重配置
manager = AdaptiveWeightManager()
weights = manager.get_weights_by_market('bull')

# 多维度评分
scoring = MultiDimensionScoring(weights)
score_result = scoring.calculate_score(
    stock_data, fundamental, technical, capital_flow
)
```
