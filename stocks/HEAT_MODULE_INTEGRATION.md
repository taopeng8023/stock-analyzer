## 📈 热度评分逻辑

### 排名分段

| 排名区间 | 百分比 | 评分 | 说明 |
|----------|--------|------|------|
| 1-55 | 前 1% | 1.0 分 | 极热，警惕出货 |
| 56-275 | 前 5% | 2.0 分 | 高度关注 ⭐ |
| 276-825 | 前 15% | 1.5 分 | 适度关注 (最佳) ⭐⭐ |
| 826-2750 | 前 50% | 1.0 分 | 关注度一般 |
| 2750+ | 后 50% | 0.5 分 | 关注度低 |

### 评分逻辑

```
为什么前 1% 只有 1 分？
- 热度极高可能是出货信号
- 散户扎堆，主力可能离场
- 警惕"热度陷阱"

为什么前 5-15% 最高分？
- 有关注度但不过热
- 主力可能在建仓/拉升
- 风险收益比最佳
```

## ⚠️ 注意事项

### 1. 热度数据特点

| 特点 | 说明 | 应对 |
|------|------|------|
| **实时变化** | 每分钟都在变 | 盘后分析更准确 |
| **滞后性** | 涨起来才热 | 结合技术面 |
| **反向信号** | 过热可能是顶 | 前 1% 降分 |
| **平台差异** | 雪球 vs 东财 | 使用雪球数据 |

### 2. 适用场景

| 场景 | 适用度 | 说明 |
|------|--------|------|
| **短线选股** | ⭐⭐⭐⭐⭐ | 发现热门股 |
| **趋势跟踪** | ⭐⭐⭐⭐ | 热度上升=资金流入 |
| **价值投资** | ⭐⭐ | 热度参考意义小 |
| **反向操作** | ⭐⭐⭐⭐ | 极热卖出，极冷买入 |

### 3. 风险提示

```
⚠️ 热度指标不能作为唯一选股依据！

必须结合:
- 技术面 (金叉/趋势)
- 资金流 (主力净流入)
- 筹码面 (集中度)
- 基本面 (业绩/估值)
```
