# ⚠️ 数据使用政策

## 真实数据原则

**本工具所有股票交易相关数据必须来自真实市场数据源，严禁使用任何模拟/伪造/估算数据。**

### 严禁估算数据

**特别强调：以下行为也被禁止**
- ❌ 基于成交额估算主力净流入
- ❌ 基于成交量估算资金流向
- ❌ 任何基于公式推算的"伪数据"
- ❌ 即使标注为"估算"也不允许

**原因**：估算数据可能误导用户，造成经济损失。

### 适用场景

此政策适用于**所有场景**，包括但不限于：
- ✅ 日常查询
- ✅ 功能测试
- ✅ 开发调试
- ✅ 演示展示
- ✅ 用户请求
- ✅ 定时任务
- ✅ 任何时间、任何情况

### 数据源要求

**允许使用的数据源：**
1. 腾讯财经（实时行情）
2. 东方财富（K 线、资金流）
3. 百度股市通（排行榜）
4. 新浪财经（行情查询）
5. 其他官方或授权数据源

**严禁使用：**
- ❌ 随机生成的模拟数据
- ❌ 人为伪造的交易数据
- ❌ 过期的缓存数据（超出合理时间）
- ❌ 任何非真实市场数据

### 技术实现

```python
# ✅ 正确：数据源失败时返回空数据
klines = fetch_eastmoney(symbol)
if not klines:
    klines = fetch_tencent(symbol)
if not klines:
    return []  # 返回空数据，不使用模拟数据

# ❌ 错误：生成模拟数据
if not klines:
    return generate_fake_data(symbol)  # 严禁！
```

### 失败处理

当所有数据源都不可用时：

1. **返回空数据** - 不生成任何模拟数据
2. **明确提示用户** - 说明数据获取失败
3. **建议稍后重试** - 网络恢复后重新获取
4. **检查缓存** - 可使用合理的缓存数据（需标注时间）

### 缓存政策

- 缓存数据必须标注时间戳
- 默认缓存有效期：行情 10 分钟，K 线 60 分钟
- 过期缓存必须标注"已过期"
- 不得使用过期缓存作为实时数据

### 用户提示

任何数据展示都必须明确标注：
- 数据来源
- 获取时间
- 是否实时
- 缓存状态

示例：
```
📈 sh600000 K 线数据
数据源：东方财富
更新时间：2026-03-17 22:55:00
状态：实时数据
```

### 违规后果

使用模拟数据可能导致：
- 误导交易决策
- 造成经济损失
- 违反法律法规
- 损害用户信任

**任何情况下都不得以"测试"、"演示"、"临时"等理由使用模拟数据。**
