---

## 🎯 功能概述

股票筛选工作流 v3.0 新增决策回溯与自动优化模块：

1. **决策记录** - 自动记录每次工作流的决策结果
2. **跟踪对比** - 跟踪后续 10 个交易日的表现
3. **回测报告** - 生成决策准确率报告
4. **自动优化** - 基于历史表现优化决策权重

## 🎯 优化机制

### 1. 评级权重优化

基于历史胜率调整评级阈值：

| 当前评级 | 胜率 >70% | 胜率 <40% |
|---------|----------|----------|
| 强烈推荐 | 保持 | 提高标准 |
| 推荐 | 保持 | 降级为谨慎推荐 |
| 谨慎推荐 | 升级为推荐 | 降级为观望 |
| 观望 | 升级为谨慎推荐 | 保持 |

### 2. 置信度阈值优化

```python
# 基于历史表现调整
if 高置信度胜率 > 75%:
    降低高置信度门槛 (75% → 70%)
elif 高置信度胜率 < 50%:
    提高高置信度门槛 (75% → 80%)
```

### 3. 评分因子优化

```python
# 原始评分
score = 成交额×0.4 + 涨幅×0.3 + 成交量×0.3

# 优化后 (基于回测)
if 成交额因子表现好:
    score = 成交额×0.5 + 涨幅×0.25 + 成交量×0.25
```

## 🔄 自动化流程

### Crontab 配置

```bash
# 每个交易日 15:10 运行工作流 (收盘后)
10 15 * * 1-5 cd /home/admin/.openclaw/workspace/stocks && python3 run_workflow.py --strategy all --top 10 --push --record

# 每个交易日 9:00 跟踪决策
0 9 * * 1-5 cd /home/admin/.openclaw/workspace/stocks && python3 run_workflow.py --backtest

# 每周五 17:00 生成周报告
0 17 * * 5 cd /home/admin/.openclaw/workspace/stocks && python3 run_workflow.py --report

# 每月 1 号 18:00 优化权重
0 18 1 * * cd /home/admin/.openclaw/workspace/stocks && python3 run_workflow.py --optimize
```

## ⚠️ 注意事项

### 1. 数据要求

- 至少需要 5 个已完成决策才能进行优化
- 每个决策需要 10 个交易日才能完成跟踪
- 建议连续运行 2 周以上再进行优化

### 2. 风险提示

- 回测结果不代表未来表现
- 优化建议仅供参考
- 不构成投资建议

### 3. 数据维护

```bash
# 清理旧数据 (保留最近 3 个月)
find backtest_data -name "*.json" -mtime +90 -delete

# 导出历史数据
python3 backtest.py --export --format csv
```
