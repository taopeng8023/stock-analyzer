## 📊 评分规则

### 基本面加分 (0-0.2)

```python
# 大盘股加分 (流动性好，风险低)
amount > 100 亿：+0.08
amount > 50 亿：+0.05

# 行业龙头加分
名称包含 ['茅台','宁德','平安','工行','建行','招行','比亚迪',...]：+0.06

# 低估值加分 (简化：股价<50 元)
0 < price < 50：+0.03
0 < price < 20：+0.03 (额外)

# 上限：0.2
```

### 机构研报加分 (0-0.2)

```python
# 高成交额代表机构关注度高
amount > 50 亿：+0.10
amount > 20 亿：+0.06
amount > 10 亿：+0.03

# 高成交量代表活跃度高
volume > 5000 万手：+0.05
volume > 2000 万手：+0.03

# 上限：0.2
```
