**新增方法**:

```python
def _calculate_tech_bonus(self, stock: dict) -> float:
    """
    计算技术面加分 (简化版)
    
    基于:
    - 涨跌幅
    - 成交量变化
    - 价格趋势
    
    返回：0-0.2 的加分
    """
```

**评分规则**:
```python
# 涨跌幅技术信号
change_pct > 7%: +0.08 (强势上涨)
change_pct > 3%: +0.05 (温和上涨)
change_pct > 0%: +0.02 (小幅上涨)

# 成交量技术信号
amount > 50 亿：+0.06 (放量)
amount > 10 亿：+0.03 (温和放量)

# 量价配合
change_pct > 3% AND volume > 200 万：+0.06 (价涨量增)

# 上限：0.2
```

**理由生成**:
```python
if tech_bonus >= 0.2:
    reasons.append('技术面强势')
elif tech_bonus >= 0.1:
    reasons.append('技术面偏强')
```
