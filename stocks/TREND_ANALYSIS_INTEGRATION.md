## 📁 新增文件

### trend_analysis_strategy.py

**技术分析维度**:

1. **道氏理论** (20% 权重)
   - 主要趋势判断
   - 趋势阶段识别

2. **均线系统** (20% 权重)
   - MA5/10/20/60
   - 多头/空头排列

3. **K 线形态** (15% 权重)
   - 反转形态
   - 整理形态
   - 持续形态

4. **支撑阻力位** (10% 权重)
   - 支撑位计算
   - 阻力位计算
   - 价格位置判断

5. **成交量确认** (15% 权重)
   - 量价关系
   - 成交量确认

6. **MACD 指标** (10% 权重)
   - 金叉/死叉
   - 背离信号

7. **KDJ 指标** (5% 权重)
   - 超买/超卖
   - 金叉/死叉

8. **RSI 指标** (5% 权重)
   - 超买/超卖
   - 强弱判断

## 📊 评分规则

### 综合评分

```python
total_score = (
    道氏理论 × 0.20 +
    均线系统 × 0.20 +
    K 线形态 × 0.15 +
    支撑阻力 × 0.10 +
    成交量确认 × 0.15 +
    MACD × 0.10 +
    KDJ × 0.05 +
    RSI × 0.05
)
```

### 决策加分

| 综合评分 | 趋势判断 | 决策加分 |
|---------|---------|---------|
| >=75 | 强烈上涨 | +0.30 |
| >=65 | 上涨 | +0.25 |
| >=55 | 偏强 | +0.20 |
| >=45 | 盘整 | +0.10 |
| >=35 | 偏弱 | +0.05 |
| <35 | 下跌 | +0.00 |
