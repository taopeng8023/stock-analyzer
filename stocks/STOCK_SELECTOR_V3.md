## 📈 选股流程

```
┌─────────────────────────────────────────────────────────┐
│  第 1 步：获取资金流前 100 只股票                          │
│  - ak.stock_fund_flow_individual(symbol='即时')         │
├─────────────────────────────────────────────────────────┤
│  第 2 步：获取历史行情                                   │
│  - data_source.get_stock_history()                      │
│  - 优先使用新浪接口 (稳定)                               │
│  - 自动缓存 (5 分钟)                                     │
├─────────────────────────────────────────────────────────┤
│  第 3 步：技术面分析                                     │
│  - 五重过滤金叉检查                                      │
│  - 波浪理论分析                                          │
│  - 江恩理论分析                                          │
├─────────────────────────────────────────────────────────┤
│  第 4 步：资金流评分                                     │
│  - 主力净流入>1 亿：+3 分                                │
│  - 主力净流入>5000 万：+2 分                             │
│  - 主力净流入>1000 万：+1 分                             │
├─────────────────────────────────────────────────────────┤
│  第 5 步：综合评分                                       │
│  - 总分>=6：强烈推荐                                    │
│  - 总分>=5：推荐                                        │
│  - 总分>=4：观望                                        │
│  - 总分<4：回避                                         │
├─────────────────────────────────────────────────────────┤
│  第 6 步：输出结果                                       │
│  - 按评分排序                                           │
│  - 保存 JSON 文件                                         │
└─────────────────────────────────────────────────────────┘
```

## 🎯 选股标准

### 强烈推荐 (评分>=6)

- ✅ 五重过滤至少 4 项通过
- ✅ 主力净流入>5000 万
- ✅ 波浪/江恩评分高

### 推荐 (评分>=5)

- ✅ 五重过滤至少 3 项通过
- ✅ 主力净流入>1000 万

### 观望 (评分>=4)

- ✅ 基础金叉通过
- ✅ 部分技术指标符合

### 回避 (评分<4)

- ❌ 不符合条件

---

## ⚠️ 注意事项

1. **限频保护** - 每 5 次请求延迟 1 秒
2. **数据缓存** - 5 分钟缓存，减少重复请求
3. **资金流时效** - 资金流数据实时变化，建议盘中使用
4. **综合判断** - 资金流只是参考，需结合技术面
