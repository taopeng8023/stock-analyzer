def _generate_final_decision(all_results, top_n):
    """
    生成最终决策:

## 🎯 筛选逻辑

### 综合评分算法

```python
评分 = 成交额得分 + 涨幅得分 + 成交量得分

成交额得分 = 成交额 (亿) × 10
涨幅得分 = max(0, 涨跌幅%) × 2
成交量得分 = 成交量 (万手) / 1000
```

### 命中策略数

- **1**: 只出现在一个策略结果中
- **2**: 出现在两个策略结果中
- **3+**: 出现在三个策略结果中 (强烈推荐)
