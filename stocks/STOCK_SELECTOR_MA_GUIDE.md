# 双均线选股器 - 五重过滤增强版

## 📋 功能说明

本选股器基于双均线金叉策略，集成五重过滤条件，有效避免伪金叉信号。

## 🎯 五重过滤条件

| 过滤 | 条件 | 说明 | 默认值 |
|------|------|------|--------|
| **1. 基础金叉** | MA5 上穿 MA20 | 必须满足 | - |
| **2. 成交量** | 当日成交量 > MA20(成交量) × 1.5 | 放量确认 | 1.5 倍 |
| **3. 趋势** | 收盘价 > MA200 | 只做多头趋势 | - |
| **4. RSI** | 50 < RSI(14) < 75 | 强势区但不超买 | 50-75 |
| **5. MACD** | MACD > Signal 且 Histogram > 0 | 双指标共振 | - |
| **6. 均线斜率** | MA5 和 MA20 向上 | 均线方向确认 | 3 日 |

## 🚀 使用方法

### 1. 扫描全市场

```bash
# 默认模式（60% 条件满足）
python3 stock_selector_ma.py --scan

# 宽松模式（40% 条件满足，信号多但质量低）
python3 stock_selector_ma.py --scan --mode loose

# 严格模式（80% 条件满足，信号少但质量高）
python3 stock_selector_ma.py --scan --mode strict

# 自定义成交量倍数
python3 stock_selector_ma.py --scan --volume 2.0
```

### 2. 检查单只股票

```bash
# 检查特变电工
python3 stock_selector_ma.py --check 600089

# 检查立讯精密
python3 stock_selector_ma.py --check 002475

# 检查美利云
python3 stock_selector_ma.py --check 000815
```

## 🎛️ 参数配置

### 过滤模式

| 模式 | 条件满足 | 适用场景 | 预期胜率 |
|------|----------|----------|----------|
| **loose** | 40% (2/5) | 牛市、激进 | 55-60% |
| **medium** | 60% (3/5) | 震荡市、平衡 ⭐ | 65-70% |
| **strict** | 80% (4/5) | 熊市、保守 | 70-75% |

|------|------|------|

| 2.0x | 明显放量 | 小盘股 |
| 2.5x+ | 巨量 | 妖股/热点 |

## ⚠️ 注意事项

1. **不是 100% 准确** - 五重过滤提高胜率但不能保证盈利
2. **需要人工复核** - 建议结合基本面、消息面判断
3. **设置止损** - 建议设置 -8% 到 -10% 止损
4. **分散投资** - 不要 All in 单只股票
5. **市场环境影响** - 熊市信号少，牛市信号多

## 📁 输出文件

扫描结果保存在：
```
cache/ma_golden_cross_YYYYMMDD_HHMM.json
```

包含字段：
- symbol: 股票代码
- price: 当前价格
- change: 涨跌幅
- score: 得分 (0-5)
- filters: 各过滤条件详情

## 🔧 代码集成

可以在其他脚本中导入使用：

```python
from stock_selector_ma import EnhancedMASelector

selector = EnhancedMASelector()

# 获取数据
data = selector.fetch_kline('sh600089')

# 检查信号
passed, details = selector.check_golden_cross(data, 'medium')

if passed:
    print('符合买入条件！')
    print(f'得分：{details["score"]}/5')
```

## 📞 常见问题

**Q: 为什么扫描不到股票？**
A: 可能原因：
- 当前市场没有金叉信号（正常）
- 过滤条件太严格（尝试 loose 模式）
- 数据获取失败（检查网络）

**Q: 五重过滤必须全部满足吗？**
A: 不必。medium 模式只需 60% (3/5) 满足即可。

**Q: 最佳参数是什么？**
A: 没有绝对最佳，建议：
- 牛市：loose + 1.3x
- 震荡市：medium + 1.5x ⭐
- 熊市：strict + 2.0x
