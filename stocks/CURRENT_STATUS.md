## 🎯 数据准确性说明

### 腾讯估算数据

**准确性评估：**

| 股票类型 | 准确性 | 说明 |
|---------|--------|------|
| 高成交量 (>500 万手) | ⭐⭐⭐⭐ | 主力参与度高，估算较准 |
| 中等成交量 (100-500 万手) | ⭐⭐⭐ | 估算基本可靠 |
| 低成交量 (<100 万手) | ⭐⭐ | 偏差可能较大 |

**算法：**
```python
if volume > 500 万手：factor = 0.18  # 18%
elif volume > 100 万手：factor = 0.15  # 15%
else: factor = 0.12  # 12%

主力净流入 = 成交额 × factor
```

**使用建议：**
- ✅ 可用于日常监控和趋势分析
- ⚠️ 不建议作为唯一交易依据
- 📊 数据已标注"估算"标识
