## 📈 技术指标详解（30 分）

### 1. MACD 指标（10 分）

**平滑异同移动平均线**，判断趋势强弱和买卖点

| 指标 | 说明 | 评分规则 |
|------|------|----------|
| DIF | 差离值（快线） | - |
| DEA | 讯号线（慢线） | - |
| MACD 柱 | 两线差值 | - |

**评分标准：**
```
基础分：5 分

金叉（DIF>DEA）：+3 分
  - 红柱放大：+2 分
死叉（DIF<DEA）：-3 分
  - 绿柱放大：-2 分

0 轴上方（多头）：+1 分
0 轴下方（空头）：-1 分

得分范围：0-10 分
```

**信号解读：**
- **金叉 +0 轴上**：强烈买入信号
- **金叉 +0 轴下**：反弹信号
- **死叉 +0 轴上**：回调信号
- **死叉 +0 轴下**：强烈卖出信号

---

### 2. KDJ 指标（10 分）

**随机指标**，判断超买超卖和短期拐点

| 指标 | 说明 | 取值范围 |
|------|------|----------|
| K 值 | 快速确认线 | 0-100 |
| D 值 | 慢速主干线 | 0-100 |
| J 值 | 超快方向线 | 0-100（可超出） |

**评分标准：**
```
基础分：5 分

K>D 金叉：+2 分（K<80 未超买再 +1 分）
K<D 死叉：-2 分（K>20 未超卖再 -1 分）

超卖区（K<20,D<20）：+2 分
超买区（K>80,D>80）：-2 分

J>100（过热）：-1 分
J<0（过冷）：+1 分

得分范围：0-10 分
```

**信号解读：**
- **K<20**：超卖区，可能反弹
- **K>80**：超买区，可能回调
- **J>100**：严重超买
- **J<0**：严重超卖

---

### 3. RSI 指标（10 分）

**相对强弱指标**，衡量多空力量对比

| 指标 | 周期 | 说明 |
|------|------|------|
| RSI6 | 6 日 | 短期强弱 |
| RSI12 | 12 日 | 中期强弱 |
| RSI24 | 24 日 | 长期强弱 |

**评分标准：**
```
基础分：5 分

RSI6<20（严重超卖）：+3 分
RSI6<30（超卖）：+2 分
RSI6>80（严重超买）：-3 分
RSI6>70（超买）：-2 分

多头排列（RSI6>RSI12>RSI24）：+2 分
空头排列（RSI6<RSI12<RSI24）：-2 分

RSI6>50 且 RSI12>50（强势区）：+1 分
RSI6<50 且 RSI12<50（弱势区）：-1 分

得分范围：0-10 分
```

**信号解读：**
- **RSI<20**：严重超卖，反弹概率大
- **RSI>80**：严重超买，回调风险高
- **多头排列**：上升趋势确立

---

### 4. 均线系统（10 分）

**移动平均线**，判断趋势和支撑压力

| 均线 | 周期 | 作用 |
|------|------|------|
| MA5 | 5 日 | 攻击线（超短线） |
| MA10 | 10 日 | 操盘线（短线） |
| MA20 | 20 日 | 辅助线（月线） |
| MA60 | 60 日 | 生命线（中线） |

**评分标准：**
```
基础分：5 分

多头排列（股价>MA5>MA10>MA20）：+3 分
空头排列（股价<MA5<MA10<MA20）：-3 分
站上 MA20：+1 分
跌破 MA20：-1 分

站上 MA60 且 MA60 向上：+2 分
跌破 MA60：-1 分

趋势多头：+1 分
趋势空头：-1 分

得分范围：0-10 分
```

**信号解读：**
- **多头排列**：最佳买入形态
- **站上生命线**：中线走好
- **跌破 MA20**：短线转弱

---

### 5. 成交量指标（10 分）

**量价关系**，验证趋势有效性

| 指标 | 说明 | 健康范围 |
|------|------|----------|
| 量比 | 当日量/5 日均量 | 1.5-3.0 |
| 换手率 | 成交股数/总股本 | 3-10% |
| 成交额 | 成交金额 | >10 亿 |

**评分标准：**
```
基础分：5 分

量比≥3（明显放量）：+3 分
量比≥2：+2 分
量比≥1：+1 分
量比≤0.5（严重缩量）：-2 分
量比≤0.8：-1 分

换手率 3-10%（健康活跃）：+2 分
换手率 10-20%（较活跃）：+1 分
换手率>20%（过度活跃）：-1 分
换手率<1%（不活跃）：-1 分

成交额≥50 亿：+1 分
成交额≥10 亿：+0.5 分
成交额<1 亿：-1 分

得分范围：0-10 分
```

**信号解读：**
- **放量上涨**：健康形态
- **缩量下跌**：可能企稳
- **天量天价**：警惕见顶

---

### 6. 价格强度（10 分）

**当日价格表现**，衡量短期动能

| 指标 | 说明 | 评分 |
|------|------|------|
| 涨跌幅 | 当日涨跌百分比 | - |
| 振幅 | 最高 - 最低/昨收 | - |
| 收盘位置 | 收盘在当日区间位置 | 0-1 |

**评分标准：**
```
基础分：5 分

涨停（≥9.5%）：+4 分
涨≥5%：+3 分
涨≥2%：+2 分
涨≥0%：+1 分
跌≤-9.5%（跌停）：-4 分
跌≤-5%：-3 分
跌≤-2%：-2 分

收盘高位（≥0.8）：+2 分
收盘中位（≥0.5）：+1 分
收盘低位（≤0.2）：-2 分

振幅 3-8%（健康）：+1 分
振幅>15%（过大）：-1 分

得分范围：0-10 分
```

## ⚠️ 注意事项

1. **技术指标滞后性**：所有指标基于历史数据，存在滞后
2. **多指标共振**：单一指标可能失真，需多指标配合判断
3. **结合大盘**：个股受大盘影响大，需结合市场环境
4. **及时更新**：技术指标实时变化，建议每日更新数据
5. **止损纪律**：再好的系统也需要严格执行止损
