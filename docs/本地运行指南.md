## ⚠️ 常见问题

### Q1: 无法获取数据

**错误：** `Connection aborted`

**解决：**
1. 检查网络连接
2. 检查防火墙设置
3. 尝试使用代理

### Q2: 数据获取慢

**原因：** API 限流

**解决：**
- 添加延时：`time.sleep(1)`
- 减少并发请求

### Q3: 回测无交易

**原因：** 评分阈值太高

**解决：**
- 降低买入阈值（75 → 70）
- 检查技术指标计算

## 📝 技术说明

### 数据来源

- **K 线数据：** 东方财富 API
- **资金流数据：** 东方财富 API
- **技术指标：** 本地计算

### 评分系统

| 维度 | 分值 | 说明 |
|------|------|------|
| 技术面 | 30 分 | MACD+KDJ+RSI+ 均线 + 成交量 + 价格 |
| 资金面 | 20 分 | 主力净流入 + 超大单占比 |
| 题材面 | 20 分 | 板块热度 + 概念催化 |
| 基本面 | 15 分 | PE+ 利润增长 + 负债率 |
| 风险面 | 15 分 | 高位风险 + 亏损状态 |
| **总计** | **100 分** | - |

### 交易策略

**买入条件：**
- 综合评分 ≥ 75 分
- 均线趋势非强势空头
- 无持仓

**卖出条件：**
- 综合评分 < 50 分
- 持仓 ≥ 7 天且评分 < 60 分
- 止盈 15%
- 止损 6%
- 追踪止损（盈利 8% 后回撤 5%）
