## 🔍 根本原因分析

### 网络环境限制

**证据：**
1. 所有外部 HTTP/HTTPS 请求都失败
2. 错误信息一致：`Connection aborted`
3. 不同数据源都失败
4. 浏览器可以访问部分网页（说明不是完全断网）

**结论：**
服务器防火墙限制了对外部金融 API 的访问

**可能原因：**
1. 公司/机构网络策略
2. 云服务器安全组配置
3. 金融 API 被加入黑名单
4.  outbound 流量限制
