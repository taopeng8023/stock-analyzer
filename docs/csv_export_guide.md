# K 线数据导出指南

## 📋 为什么要手动导出？

由于网络环境限制，浏览器自动化无法直接获取数据。我们需要手动从东方财富导出 CSV，然后进行回测。

**【重要】这是获取真实数据的可靠方式**

---

## 📊 导出步骤

### 方法一：东方财富（推荐）

1. **访问个股页面**
   ```
   https://quote.eastmoney.com/{股票代码}.html
   例如：https://quote.eastmoney.com/600549.html
   ```

2. **查看历史交易**
   - 点击页面中间的"历史交易"标签
   - 选择时间范围（建议 1 年以上）

3. **导出数据**
   - 点击"导出数据"按钮
   - 选择 CSV 格式
   - 保存到：`/home/admin/.openclaw/workspace/data/kline/{股票代码}.csv`

---

### 方法二：同花顺

1. **访问 iFinD**
   ```
   http://data.10jqka.com.cn/
   ```

2. **搜索股票代码**

3. **导出历史数据**
   - 选择"历史成交"
   - 导出 CSV

---

## 📁 CSV 文件格式

### 必需列名

CSV 文件必须包含以下列（列名可以是中文或英文）：

| 中文列名 | 英文列名 | 类型 | 说明 |
|---------|---------|------|------|
| 日期 | date | string | YYYY-MM-DD |
| 开盘 | open | float | 开盘价 |
| 最高 | high | float | 最高价 |
| 最低 | low | float | 最低价 |
| 收盘 | close | float | 收盘价 |
| 成交量 | volume | int | 成交量（手） |
| 成交额 | amount | float | 成交额（元）可选 |

### 示例 CSV

```csv
日期，开盘，最高，最低，收盘，成交量，成交额
2025-03-25,18.50,19.20,18.30,19.00,125000,237500000
2025-03-24,18.20,18.60,18.00,18.50,98000,179800000
2025-03-23,18.00,18.40,17.80,18.20,110000,199800000
```

## 📊 资金流数据（可选）

如需更精确的回测，可以导出资金流数据：

1. **访问**：https://data.eastmoney.com/zjlx/{股票代码}.html
2. **复制最新数据**：
   ```json
   {
     "main_net_inflow": 29690,
     "main_ratio": 22.76,
     "super_ratio": 27.70
   }
   ```
3. **保存**：`/home/admin/.openclaw/workspace/data/fund_flow/{股票代码}.json`

## 📝 常见问题

### Q1: 导出的 CSV 无法识别？

**A:** 检查列名是否匹配，支持中英文：
- 日期 / date
- 开盘 / open
- 最高 / high
- 最低 / low
- 收盘 / close
- 成交量 / volume

### Q2: 数据量不够？

**A:** 在东方财富选择更长的时间范围（建议 1-2 年）

### Q3: 回测结果为空？

**A:** 确保 CSV 至少有 60 条数据（约 3 个月）
