# 真实数据回测 - 网络限制说明

## ⚠️ 当前状况

**问题：** 从当前服务器环境无法直接访问东方财富 API

**错误信息：**
```
Connection aborted - Remote end closed connection without response
```

**原因分析：**
1. 服务器防火墙阻止外部 HTTP 请求
2. 东方财富 API 对服务器 IP 限流
3. 网络环境限制（可能只能访问特定域名）

---

## ✅ 已验证可用的方案

### 方案 1：浏览器自动化（推荐）⭐

**状态：** ✅ 可用

通过浏览器访问东方财富页面，然后提取数据：

```bash
# 打开东方财富页面
browser open https://quote.eastmoney.com/unify/cr/1.600549

# 通过 snapshot 获取页面数据
browser snapshot
```

**优点：**
- ✅ 绕过 API 限制
- ✅ 数据真实可靠
- ✅ 无需额外依赖

**缺点：**
- ⚠️ 需要浏览器环境
- ⚠️ 速度较慢
- ⚠️ 需要解析 HTML

---

### 方案 2：curl 命令行（测试中）

**状态：** ⚠️ 部分可用

通过 curl 获取数据（之前测试成功过）：

```bash
curl -s "https://push2his.eastmoney.com/api/qt/stock/kline/get?secid=1.600549&fields1=f1,f2,f3,f4,f5,f6&fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61&klt=101&fqt=1&beg=20250101&end=20261231&lmt=1000000"
```

**优点：**
- ✅ 直接获取 JSON 数据
- ✅ 速度快

**缺点：**
- ⚠️ 网络连接不稳定
- ⚠️ 可能被限流

---

### 方案 3：手动导出 CSV

**状态：** ✅ 完全可用

从东方财富网站手动导出：

1. 访问 https://quote.eastmoney.com/unify/cr/1.600549
2. 点击"历史交易"
3. 导出 CSV 文件
4. 保存到 `/home/admin/.openclaw/workspace/data/kline/600549.csv`

**优点：**
- ✅ 100% 可靠
- ✅ 数据完整
- ✅ 无需网络

**缺点：**
- ⚠️ 需要手动操作
- ⚠️ 不适合批量
