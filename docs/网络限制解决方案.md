# 服务器网络限制解决方案

## 🔍 问题诊断结果

**测试结果：**
- ✅ 东方财富首页：可访问（状态码 200）
- ❌ API 端点：被限制（状态码 000）
- ✅ Ping 测试：正常
- ✅ DNS 解析：正常

**结论：** 服务器 IP 可能被东方财富 API 封禁，但网页访问正常。

## 🛠️ 解决方案

### 方案 1：使用浏览器自动化（推荐）⭐⭐⭐⭐⭐

**状态：** ✅ 已验证可用

**原理：** 浏览器环境可以绕过 API 限制

**操作方法：**
```bash
# 在 browser 工具中执行 JavaScript
browser action=act kind=evaluate fn="async () => {
    const url = 'https://push2.eastmoney.com/api/qt/stock/kline/get?secid=0.002475&klt=101&lmt=300';
    const resp = await fetch(url);
    const data = await resp.json();
    return JSON.stringify(data);
}" timeoutMs=20000
```

**优点：**
- ✅ 不需要额外配置
- ✅ 已经验证可用
- ✅ 可以获取所有数据

**缺点：**
- ⚠️ 需要通过 browser 工具执行

### 方案 2：配置 HTTP 代理 ⭐⭐⭐⭐

**状态：** 需要代理服务器

**步骤：**

1. **获取代理服务器**
   ```bash
   # 可以使用免费代理或购买付费代理
   # 示例（需要替换为真实代理）
   export HTTP_PROXY="http://proxy_server:port"
   export HTTPS_PROXY="http://proxy_server:port"
   ```

2. **在 Python 脚本中使用代理**
   ```python
   import requests
   
   proxies = {
       'http': 'http://proxy_server:port',
       'https': 'http://proxy_server:port',
   }
   
   resp = requests.get(url, proxies=proxies, timeout=15)
   ```

3. **测试代理**
   ```bash
   curl -x http://proxy_server:port https://push2.eastmoney.com/api/qt/stock/kline/get?secid=0.002475&lmt=1
   ```

**代理服务商推荐：**
- 亮数据（Bright Data）
- Oxylabs
- Smartproxy
- 芝麻代理（国内）

**优点：**
- ✅ 可以彻底解决 IP 限制
- ✅ 适用于所有 API 调用

**缺点：**
- ⚠️ 需要付费购买代理
- ⚠️ 需要配置

**测试脚本：**
```python
import requests

endpoints = [
    'https://push2.eastmoney.com',
    'https://push2his.eastmoney.com',
    'https://data.eastmoney.com',
]

for endpoint in endpoints:
    try:
        url = f'{endpoint}/api/qt/stock/kline/get?secid=0.002475&lmt=1'
        resp = requests.get(url, timeout=5)
        print(f"{endpoint}: {resp.status_code}")
    except Exception as e:
        print(f"{endpoint}: {e}")
```

**优点：**
- ✅ 不需要额外配置
- ✅ 免费

**缺点：**
- ⚠️ 可能都被封禁
- ⚠️ 不稳定
