## 📊 数据说明

### 主力净流入定义

**主力净流入 = 超大单净流入 + 大单净流入**

| 订单类型 | 定义 |
|---------|------|
| 超大单 | 单笔成交≥100 万元 或 ≥5000 手 |
| 大单 | 单笔成交≥20 万元 或 ≥500 手 |
| 中单 | 单笔成交≥4 万元 或 ≥100 手 |
| 小单 | 单笔成交<4 万元 或 <100 手 |

### 主力净占比

```
主力净占比 = 主力净流入 / 成交额 × 100%
```

**意义：**
- >10%：主力大幅流入
- 5-10%：主力流入
- 0-5%：主力小幅流入
- <0%：主力流出
