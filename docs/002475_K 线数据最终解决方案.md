### 推荐方案：使用 AKShare

**在本地电脑运行：**

```bash
# 1. 安装 AKShare
pip3 install akshare pandas

# 2. 创建脚本
cat > get_kline.py << 'EOF'
import akshare as ak
import pandas as pd

print("📡 正在获取 002475 立讯精密 K 线数据...")

# 获取历史行情
df = ak.stock_zh_a_hist(
    symbol="002475",
    period="daily",
    start_date="20250325",
    end_date="20260325",
    adjust="qfq"
)

print(f"✅ 获取到 {len(df)} 条数据")
print(f"\n最新 10 条数据：")
print(df.tail(10).to_string())

# 保存
df.to_csv("002475_kline.csv", index=False, encoding='utf-8-sig')
print(f"\n✅ 数据已保存至：002475_kline.csv")

# 统计
print(f"\n📊 统计信息")
print(f"最高价：{df['最高'].max():.2f} 元")
print(f"最低价：{df['最低'].min():.2f} 元")
print(f"最新价：{df['收盘'].iloc[-1]:.2f} 元")
print(f"区间涨幅：{(df['收盘'].iloc[-1] / df['收盘'].iloc[0] - 1) * 100:.2f}%")
EOF

# 3. 运行
python3 get_kline.py
```

**预期时间：** 10-30 秒
