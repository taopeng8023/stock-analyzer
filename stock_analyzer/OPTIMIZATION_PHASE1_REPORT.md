## ✅ 优化内容

### 1️⃣ 数据源优化

**优化前**: 单一数据源（东方财富）
**优化后**: 多数据源 fallback 机制

#### 支持的数据源

| 优先级 | 数据源 | 状态 | 说明 |
|--------|--------|------|------|
| 1 | 东方财富 | ✅ | 主数据源，数据最全 |
| 2 | 新浪财经 | ✅ | 备用数据源，稳定 |
| 3 | 同花顺 | 🔄 | 准备中 |

#### 技术实现

```python
class MultiSourceDataFetcher:
    def fetch_with_fallback(self, stock_code: str) -> dict:
        # 优先东方财富
        data = self.fetch_from_eastmoney(stock_code)
        if data and data.get('price', 0) > 0:
            return data
        
        # 备用新浪
        data = self.fetch_from_sina(stock_code)
        if data and data.get('price', 0) > 0:
            return data
        
        return {}
```

**优势**:
- ✅ 自动切换数据源
- ✅ 提高数据获取成功率
- ✅ 减少 API 失败影响

### 2️⃣ 主力资金分析集成

**新增模块**: `MoneyFlowAnalyzer`

#### 功能

1. **个股资金流向分析**
   - 主力净流入金额
   - 主力净流入占比
   - 超大单/大单/中单/小单分布

2. **主力排名获取**
   - 今日主力净流入排名
   - 主力净流入占比排名
   - TOP 20 股票池

3. **资金面评分**
   - 根据主力净流入评分
   - 根据主力占比评分
   - 综合评级（很强/较强/中等/较弱/很弱）

#### 评分标准

| 主力净流入 | 评分加成 | 主力占比 | 评分加成 |
|-----------|---------|---------|---------|
| > 1 亿 | +20 | > 10% | +15 |
| > 0.5 亿 | +15 | > 5% | +10 |
| > 0 | +5 | > 0% | +5 |
| < -1 亿 | -20 | < 0% | 0 |
| < -0.5 亿 | -15 | - | - |

### 3️⃣ 评分系统优化

**优化权重配置**

#### 优化前
```
技术面：40%
基本面：35%
资金面：25%
```

#### 优化后
```
技术面：35%  (-5%)
基本面：35%  (不变)
资金面：30%  (+5%)
```

**理由**:
- 增加资金面权重，更注重大资金动向
- 技术面权重略降，避免过度技术化
- 基本面保持，价值投资是核心

#### 综合评分计算

```python
total_score = (
    tech_score * 0.35 +      # 技术面
    fund_score * 0.35 +      # 基本面
    money_score * 0.30       # 资金面（新增）
)
```

## 📊 成功概率计算优化

### 优化前
```
基础概率：50%
评分加成：(总分 -50) * 0.6
```

### 优化后
```
基础概率：50%
评分加成：(总分 -50) * 0.5
资金加成：(资金分 -50) * 0.2
```

**新增资金面因子**，让主力资金流向直接影响成功概率。
