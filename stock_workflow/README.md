**版本**: 1.1  
**创建时间**: 2026-03-19  
**状态**: ✅ 已完成

---

## 📋 项目概述

本工作流通过整合基本面、技术面、市场情绪等多维度数据，结合机器学习模型预测个股未来三个交易日的收益表现，为投资者提供高质量的候选股票列表。

**核心价值**：从"数据收集"到"决策输出"的完整闭环，确保投资决策的科学性和可操作性。

---

## 🏗️ 系统架构

```
[股票筛选] → [基本面] → [技术分析] → [市场消息] → [数据质量监控] → [决策融合] → [结果输出]
```

### 7 大核心模块

| 模块 | 文件 | 功能 |
|------|------|------|
| 股票筛选 | `modules/stock_filter.py` | 主板 + 成交量放大筛选 |
| 基本面分析 | `modules/fundamental.py` | PE/PB/ROE 等财务指标 |
| 技术分析 | `modules/technical.py` | 15 个技术指标计算 |
| 市场消息 | `modules/news_sentiment.py` | 新闻情感量化分析 |
| 数据质量 | `modules/data_quality.py` | 三级质量监控 |
| 决策融合 | `modules/decision_fusion.py` | XGBoost 预测模型 |
| 结果输出 | `modules/output.py` | 多格式输出 + 推送 |

## ⚙️ 配置说明

### 股票筛选配置

```ini
[STOCK_FILTER]
# 主板筛选（60/00 开头）
main_board_prefix = 60, 00

# 排除板块（创业板/科创板等）
exclude_prefix = 300, 688, 8, 2, 4

# 成交量放大倍数
volume_amplify_ratio = 1.5
```

### 模型配置

```ini
[MODEL]
# 训练窗口（交易日）
train_window = 250

# 预测周期（交易日）
predict_horizon = 3

# 推荐股票数量
top_n_stocks = 10
```

## 📊 数据流

### 每日工作流程

| 时间 | 阶段 | 说明 |
|------|------|------|
| 15:30-15:45 | 股票筛选 | 获取全市场股票 + 成交量筛选 |
| 15:45-16:00 | 特征提取 | 基本面/技术/消息并行处理 |
| 16:00-16:15 | 质量监控 | 数据质量检查 + 异常处理 |
| 16:15-16:30 | 决策融合 | 模型预测 + 排序 |
| 16:30-16:45 | 结果输出 | 格式化 + 推送 |

## 🎯 核心特性

### 1. 成交量放大筛选

```python
# T 日成交量 > 1.5 × 5 日均量
volume_ratio = vol_t / volume_ma5
if volume_ratio > 1.5:
    保留股票
```

### 2. 三级数据质量监控

| 级别 | 检查项 | 阈值 |
|------|--------|------|
| 完整性 | 缺失率 | <10% |
| 有效性 | 异常值 | 1%-99% 分位数 |
| 一致性 | 异常率 | <5% |

### 3. XGBoost 决策模型

- 输入：基本面 + 技术 + 消息特征
- 输出：上涨概率
- 排序：按概率从高到低
- 输出：Top10 推荐

## ⚠️ 风险提示

1. **数据质量**：本系统依赖历史数据，数据质量直接影响预测结果
2. **模型风险**：机器学习模型存在过拟合风险，需定期重新训练
3. **市场风险**：股市有风险，投资需谨慎，本报告仅供参考
4. **时效性**：预测基于 T 日数据，市场变化可能导致结果失效
