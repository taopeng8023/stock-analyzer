# A 股股票分析技能

本技能提供 A 股市场的综合分析，帮助生成买卖决策信号。

## 核心功能

1. **实时行情查询** - 东方财富 API 实时数据
2. **技术指标计算** - MA、MACD、RSI、KDJ、布林带
3. **缠论分析** - 分型、笔、线段、中枢、背驰、三类买卖点
4. **蜡烛图分析** - 识别 50+ 种经典 K 线形态
5. **资金流向** - 主力/超大单/大单/中单/小单净流入
6. **机构研报** - 最新机构评级和研究报告
7. **公告信息** - 公司最新公告
8. **买卖信号生成** - 综合多项指标给出投资建议
9. **策略回测** - 双均线金叉 + 移动止盈回测系统

## 使用方法

### 基础分析

```bash
python scripts/analyze_stock.py <股票代码>
```

### 缠论分析

```bash
python scripts/analyze_stock.py 000001 --chanlun
```

### 蜡烛图分析

```bash
python scripts/analyze_stock.py 000001 --candle
```

识别的蜡烛图形态：
- **单根**: 锤头线、射击之星、十字星、上吊线等
- **双根**: 吞没形态、孕线、乌云盖顶、刺透形态
- **三根**: 早晨之星、黄昏之星、红三兵、三只乌鸦

### 完整报告

```bash
python scripts/analyze_stock.py 000001 --full
```

包含：缠论 + 蜡烛图 + 资金流 + 公告 + 研报

### 策略回测

```bash
# 基础回测（双均线金叉 + 移动止盈）
python scripts/backtest_ma.py 000001.SZ --token <你的 Tushare Token>

# 自定义参数
python scripts/backtest_ma.py 600519.SH \
  --short-ma 10 \
  --long-ma 30 \
  --profit-stop 0.15 \
  --start-date 20240101 \
  --end-date 20260329 \
  --capital 200000 \
  --token <你的 Tushare Token>

# 导出交易记录和资金曲线
python scripts/backtest_ma.py 000001.SZ --export --token <你的 Tushare Token>
```

**回测策略说明：**
- **买入信号**：短期均线上穿长期均线（金叉）
- **卖出信号**：移动止盈触发 或 短期均线下穿长期均线（死叉）
- **移动止盈**：从持仓后最高价回撤超过设定阈值时自动止盈

**回测报告包含：**
- 收益指标：总收益率、年化收益率
- 风险指标：最大回撤、波动率、夏普比率
- 交易指标：交易次数、胜率、盈亏比

## 缠论 + 蜡烛图结合

### 最佳实践

1. **分型确认**
   - 顶分型 + 射击之星/黄昏之星 → 强烈卖出
   - 底分型 + 锤头线/早晨之星 → 强烈买入

2. **笔的确认**
   - 向上笔起点 + 看涨蜡烛组合 → 确认买入
   - 向下笔起点 + 看跌蜡烛组合 → 确认卖出

3. **背驰确认**
   - 底背驰 + 看涨蜡烛 → 一买确认
   - 顶背驰 + 看跌蜡烛 → 一卖确认

4. **中枢突破**
   - 突破中枢 + 大阳线 → 确认突破
   - 跌破中枢 + 大阴线 → 确认跌破
