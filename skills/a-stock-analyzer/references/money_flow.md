## M01: 主力资金追踪

### 一、资金分类

**按订单大小分类**:

| 类型 | 单笔金额 | 特点 | 意义 |
|------|---------|------|------|
| **超大单** | >100 万 | 机构/主力 | 强烈信号 |
| **大单** | 20-100 万 | 大户/游资 | 重要信号 |
| **中单** | 4-20 万 | 中户 | 参考 |
| **小单** | <4 万 | 散户 | 反向指标 |

---

### 二、主力资金净流入

**公式**:
```
主力净流入 = 超大单净流入 + 大单净流入
主力净流入占比 = 主力净流入 / 总成交额 × 100%
```

**判断标准**:

| 主力净流入占比 | 含义 | 操作 |
|---------------|------|------|
| > 10% | 主力大幅流入 | 强烈关注 |
| 5-10% | 主力流入 | 关注 |
| -5% ~ 5% | 主力平衡 | 观望 |
| -10% ~ -5% | 主力流出 | 警惕 |
| < -10% | 主力大幅流出 | 卖出 |

---

### 三、资金流向与股价关系

#### 1. 正常情况

| 股价 | 主力净流入 | 含义 | 操作 |
|------|-----------|------|------|
| ↗ | 流入 | 健康上涨 | 持股 |
| ↗ | 流出 | 拉高出货 | 警惕 |
| ↘ | 流入 | 主力吸筹 | 关注 |
| ↘ | 流出 | 主力出货 | 卖出 |

---

#### 2. 异常情况

```
背离情况 1: 股价下跌 + 主力大幅流入

股价：↘↘↘
主力：↗↗↗ (大幅流入)
      ↑
  主力吸筹 → 底部临近 → 关注

背离情况 2: 股价上涨 + 主力大幅流出

股价：↗↗↗
主力：↘↘↘ (大幅流出)
      ↑
  拉高出货 → 顶部临近 → 卖出
```

---

### 四、资金流向趋势

#### 1. 连续流入

```
连续 3 日主力流入:

日期  主力净流入  股价
D1    +5000 万   ↗
D2    +8000 万   ↗
D3    +1.2 亿    ↗
      ↑
  主力持续建仓 → 强烈看涨
```

**操作**:
- 连续 3 日流入：建仓
- 连续 5 日流入：加仓
- 连续 10 日流入：重仓

---

#### 2. 连续流出

```
连续 3 日主力流出:

日期  主力净流入  股价
D1    -5000 万   ↘
D2    -8000 万   ↘
D3    -1.2 亿    ↘
      ↑
  主力持续出货 → 强烈看跌
```

**操作**:
- 连续 3 日流出：减仓
- 连续 5 日流出：清仓
- 连续 10 日流出：远离

---

### 五、主力资金成本

**估算方法**:
```python
# 主力成本估算
def 主力成本计算 (资金流向数据):
    总买入金额 = 0
    总买入股数 = 0
    
    for 交易日 in 近 20 日:
        if 主力净流入 > 0:
            总买入金额 += 主力净流入
            总买入股数 += 成交量 × 主力占比
    
    主力成本 = 总买入金额 / 总买入股数
    return 主力成本

# 操作策略
if 当前股价 < 主力成本 * 0.9:
    return "股价低于主力成本 10%，主力被套，关注"
elif 当前股价 > 主力成本 * 1.5:
    return "股价高于主力成本 50%，主力获利丰厚，警惕"
```

## M02: 北向资金分析

### 一、北向资金基础

**定义**: 通过沪股通/深股通进入 A 股的外资

**特点**:
- 投资理念成熟
- 偏好价值投资
- 持股周期长
- 被称为"聪明钱"

---

### 二、北向资金数据

**查看渠道**:
- 东方财富网
- 港交所披露易
- 各大财经网站

**数据类型**:
| 类型 | 说明 | 频率 |
|------|------|------|
| 净流入额 | 当日净买入金额 | 每日 |
| 持股比例 | 持股占总股本比例 | 每日 |
| 持股市值 | 持股总市值 | 每日 |
| 十大活跃股 | 当日成交最活跃股票 | 每日 |

---

### 三、北向资金选股偏好

#### 1. 行业偏好

| 行业 | 北向持股比例 | 偏好度 |
|------|-------------|--------|
| 食品饮料 | 高 | ⭐⭐⭐⭐⭐ |
| 医药生物 | 高 | ⭐⭐⭐⭐⭐ |
| 家用电器 | 高 | ⭐⭐⭐⭐ |
| 银行 | 中 | ⭐⭐⭐ |
| 电子 | 中 | ⭐⭐⭐ |
| 化工 | 低 | ⭐⭐ |

---

#### 2. 个股特征

**北向资金偏好**:
- ✅ 行业龙头
- ✅ 业绩稳定增长
- ✅ ROE > 15%
- ✅ 毛利率高
- ✅ 现金流好
- ✅ 分红稳定
- ❌ 避免题材炒作
- ❌ 避免业绩波动大

---

### 四、北向资金信号

#### 1. 单日大幅流入

```
北向资金单日净流入 > 10 亿:

北向：+15 亿  ← 异常值
      ↑
  强烈看好 → 跟随买入
```

**操作**:
- > 10 亿：强烈关注
- 5-10 亿：关注
- < 5 亿：正常

---

#### 2. 连续流入

```
北向资金连续 5 日流入:

D1: +3 亿
D2: +5 亿
D3: +8 亿
D4: +6 亿
D5: +10 亿
    ↑
持续看好 → 中线布局
```

---

#### 3. 北向持仓变化

```
北向持股比例持续上升:

持股比例: 3% → 5% → 7% → 9%
          ↑
      持续加仓 → 长期看好
```

**操作**:
- 持股比例上升：跟随
- 持股比例下降：警惕
- 持股比例 > 10%：高度关注

---

### 五、北向 + 主力共振

**最强烈信号**:

```
北向资金 + 主力资金 同时大幅流入:

北向：+8 亿  ← 外资看好
主力：+2 亿  ← 内资看好
      ↑
  内外资共振 → 强烈买入信号
```

**胜率**: > 75%

## M03: 龙虎榜解读

### 一、龙虎榜基础

**上榜条件** (满足其一即可):
- 日涨跌幅偏离值±7%
- 日换手率>20%
- 日价格振幅>15%
- 连续 3 日涨跌幅偏离值累计±20%

---

### 二、龙虎榜数据

**包含信息**:
| 项目 | 说明 |
|------|------|
| 买入前五 | 买入金额最大的 5 个营业部 |
| 卖出前五 | 卖出金额最大的 5 个营业部 |
| 买入总额 | 前五买入合计 |
| 卖出总额 | 前五卖出合计 |
| 净额 | 买入 - 卖出 |

---

### 三、营业部分析

#### 1. 知名游资营业部

| 营业部 | 地域 | 风格 | 特点 |
|--------|------|------|------|
| 中信上海溧阳路 | 上海 | 打板 | 手法凶悍 |
| 光大杭州庆春路 | 杭州 | 趋势 | 持股周期长 |
| 国泰君安上海江苏路 | 上海 | 打板 | 连板偏好 |
| 中信北京总部 | 北京 | 综合 | 机构席位 |
| 华泰深圳益田路 | 深圳 | 短线 | 快进快出 |

---

#### 2. 机构席位

**特征**:
- 名称含"机构专用"
- 买入金额大
- 持股周期长
- 偏好价值股

**信号意义**:
```
机构大额买入 (>5000 万):
机构专用  买入 1.2 亿
      ↑
  机构看好 → 中线机会

机构大额卖出 (>5000 万):
机构专用  卖出 8000 万
      ↑
  机构不看好 → 警惕
```

---

### 四、龙虎榜分析技巧

#### 1. 买卖力量对比

```
买方力量 >> 卖方力量:

买一：中信上海  5000 万
买二：光大杭州  3000 万
买三：机构专用  4000 万
买四：...
买五：...
买入合计：1.8 亿

卖一：...
卖二：...
卖三：...
卖四：...
卖五：...
卖出合计：5000 万

净额：+1.3 亿  ← 买方占绝对优势
      ↑
  强烈看涨 → 次日跟进
```

---

#### 2. 游资接力

```
多路游资共同买入:

买一：中信上海  3000 万
买二：光大杭州  2500 万
买三：华泰深圳  2000 万
      ↑
  游资合力 → 可能连板
```

**操作**:
- 2-3 路知名游资：关注
- 4-5 路游资：强烈关注
- 游资 + 机构：最佳

---

#### 3. 一家独大

```
单一营业部买入过多:

买一：某营业部  1 亿
买二：...      1000 万
买三：...      800 万
      ↑
  一家独大 → 次日可能砸盘 → 警惕
```

**风险**:
- 买入过于集中
- 次日容易砸盘
- 不宜追高

---

### 五、龙虎榜实战策略

#### 1. 打板策略

```
条件:
✓ 涨停板
✓ 买入前五 > 卖出前五 × 2
✓ 有知名游资
✓ 无一家独大
✓ 题材热点

操作:
- 次日高开 < 3%：可跟进
- 次日高开 3-7%：观察
- 次日高开 > 7%：不追
```

---

#### 2. 机构买入策略

```
条件:
✓ 机构专用买入
✓ 买入金额 > 5000 万
✓ 基本面良好
✓ 估值合理

操作:
- 中线布局
- 分批建仓
- 持股周期 1-3 月
```

---

#### 3. 游资接力策略

```
条件:
✓ 2-3 路知名游资
✓ 买入均匀
✓ 题材热点
✓ 连板潜力

操作:
- 次日跟进
- 设置止损
- 快进快出
```

## 资金流向综合分析

### 一、多资金源共振

**最强烈信号**:

```
主力 + 北向 + 龙虎榜 同时看好:

主力净流入：+2 亿   ← 内资主力
北向资金：  +1 亿   ← 外资
龙虎榜：    机构买入 5000 万  ← 机构
            游资合力 3000 万  ← 游资
            ↑
        四方共振 → 强烈买入
```

**胜率**: > 80%

### 三、资金流向陷阱

#### 1. 对倒出货

```
表面: 主力大幅流入
实际: 左手倒右手

特征:
- 成交量异常放大
- 股价不涨
- 大单买入但股价下跌
      ↑
  对倒出货 → 卖出
```

---

#### 2. 尾盘偷袭

```
尾盘突然大单拉升:

14:50 前：股价平稳
14:55:  突然大单买入
15:00:  股价拉升 3%
      ↑
  尾盘做盘 → 次日可能低开
```

**操作**: 不追高，次日观察
