## T01: 均线系统精讲

### 一、均线基础

**常用均线周期**:

| 均线 | 周期 | 名称 | 用途 |
|------|------|------|------|
| MA5 | 5 日 | 周线 | 超短线 |
| MA10 | 10 日 | 半月线 | 短线 |
| MA20 | 20 日 | 月线 | 短中线 |
| MA60 | 60 日 | 季线 | 中线 |
| MA120 | 120 日 | 半年线 | 中长线 |
| MA250 | 250 日 | 年线 | 长线 |

---

### 二、均线排列

#### 1. 多头排列 (强烈看涨)

```
MA5 > MA10 > MA20 > MA60 > MA120
  ↑
最强
```

**特征**:
- 短期均线在长期均线上方
- 所有均线向上发散
- 股价在 MA5 上方运行

**操作**: 持股待涨，回调买入

---

#### 2. 空头排列 (强烈看跌)

```
MA5 < MA10 < MA20 < MA60 < MA120
  ↑
最弱
```

**特征**:
- 短期均线在长期均线下方
- 所有均线向下发散
- 股价在 MA5 下方运行

**操作**: 持币观望，反弹卖出

---

### 三、葛兰碧八大法则

#### 买入信号 (4 个)

```
1. 均线走平转上，股价突破均线 → 买入
   ═════ → ╱
           ↑ 买

2. 股价跌破均线，但均线仍向上 → 买入
   ╱  ↘   ╱
      ↑ 买

3. 股价回调不破均线 → 买入
   ╱  ↘
   ════↑ 买

4. 股价暴跌远离均线 → 买入 (超跌反弹)
      ╱
   ↘↓↓
     ↑ 买
```

#### 卖出信号 (4 个)

```
1. 均线走平转下，股价跌破均线 → 卖出
   ═════ → ╲
           ↑ 卖

2. 股价突破均线，但均线仍向下 → 卖出
   ╲  ↗   ╲
      ↑ 卖

3. 股价反弹不过均线 → 卖出
   ╲  ↗
   ════↑ 卖

4. 股价暴涨远离均线 → 卖出 (超买回调)
      ╲
   ↗↑↑
     ↑ 卖
```

---

### 四、均线组合策略

#### 双均线策略

```python
# 金叉买入
if MA5 上穿 MA20:
    signal = "买入"

# 死叉卖出
if MA5 下穿 MA20:
    signal = "卖出"
```

#### 三均线策略

```python
# 多头确认
if MA5 > MA10 > MA20 and 股价 > MA5:
    signal = "强烈买入"

# 空头确认
if MA5 < MA10 < MA20 and 股价 < MA5:
    signal = "强烈卖出"
```

## T02: MACD 深度应用

### 一、MACD 基础回顾

**组成**:
- **DIF (快线)**: EMA12 - EMA26
- **DEA (慢线)**: DIF 的 9 日 EMA
- **MACD 柱**: (DIF - DEA) × 2

---

### 二、MACD 背离

#### 1. 顶背离 (看跌)

```
价格：新高 ────↗ A 点
            ╱
           ╱
价格：次高 ──↗ B 点
         
MACD: 前高 ████ A 点
            ↓
MACD: 后高 ██ B 点
         ↑
     背离确认 → 卖出
```

**特征**:
- 价格创新高
- MACD 不创新高
- DIF 高度降低

**操作**: 减仓或卖出

---

#### 2. 底背离 (看涨)

```
价格：前低 ──↘ A 点
         
价格：新低 ────↘ B 点
            ╲
             ╲

MACD: 前低 ████ A 点
            ↓
MACD: 后低 ██ B 点
         ↑
     背离确认 → 买入
```

**特征**:
- 价格创新低
- MACD 不创新低
- DIF 高度抬高

**操作**: 建仓或买入

---

### 三、MACD 零轴战法

#### 1. 零轴上方 (多头市场)

```
零轴上方金叉：
    DIF ╱╲╱
    DEA ╱ ╲
       ╱   ↑ 金叉 (强烈买入)
───────零轴───────

零轴上方死叉：
    DIF ╱╲
    DEA ╱ ╲╲
       ╱   ↑ 死叉 (回调卖出)
───────零轴───────
```

**操作**:
- 零轴上金叉：强烈买入
- 零轴上死叉：回调卖出

---

#### 2. 零轴下方 (空头市场)

```
零轴下方金叉：
───────零轴───────
    DIF ╲╱╲
    DEA ╲ ╱
       ╲   ↑ 金叉 (弱势反弹)

零轴下方死叉：
───────零轴───────
    DIF ╲╱
    DEA ╲ ╱╲
       ╲   ↑ 死叉 (强烈卖出)
```

**操作**:
- 零轴下金叉：弱势反弹，小仓位
- 零轴下死叉：强烈卖出

---

### 四、MACD 双指标法

**原理**: 使用两个不同周期的 MACD

```
快 MACD(6,13,5): 敏感，先行信号
慢 MACD(12,26,9): 稳定，确认信号

买入条件:
- 快 MACD 金叉
- 慢 MACD 确认
- 双 MACD 同时红柱

卖出条件:
- 快 MACD 死叉
- 慢 MACD 确认
- 双 MACD 同时绿柱
```

## T03: KDJ/RSI 进阶

### 一、KDJ 进阶应用

#### 1. KDJ 钝化处理

**问题**: 强势行情中 KDJ 长期超买/超卖

```
强势上涨:
K: 90 → 95 → 92 → 98 → 95
   ↑ 超买区钝化，不卖出

正确操作:
- KDJ > 80 不急于卖出
- 等待 K 下穿 D 确认
- 结合趋势判断
```

**解决方案**:
- 放大周期 (日线→周线)
- 结合趋势指标 (MACD/均线)
- 等待明确死叉

---

#### 2. KDJ 周期选择

| 周期 | KDJ 参数 | 适用 |
|------|---------|------|
| 超短线 | (6,3,3) | 1-3 天 |
| 短线 | (9,3,3) | 3-10 天 |
| 中线 | (18,3,3) | 10-30 天 |
| 长线 | (24,3,3) | 1 月以上 |

---

### 二、RSI 进阶应用

#### 1. RSI 多周期比较

```
RSI(6): 55  ← 短线
RSI(12): 58  ← 中线
RSI(24): 52  ← 长线

判断:
- 短 > 中 > 长：强势
- 短 < 中 < 长：弱势
- 交错：盘整
```

---

#### 2. RSI 背离

```
顶背离:
价格：新高 ↗
RSI: 次高 ↗ (不创新高)
     ↑ 卖出

底背离:
价格：新低 ↘
RSI: 次低 ↘ (不创新低)
     ↑ 买入
```

## T04: 布林带实战

### 一、布林带基础

**组成**:
- **上轨 (UP)**: MB + 2×标准差
- **中轨 (MB)**: 20 日均线
- **下轨 (DOWN)**: MB - 2×标准差

---

### 二、布林带收口/开口

#### 1. 收口 (变盘前兆)

```
    ╲        ╱
     ╲      ╱
      ╲    ╱
       ╲  ╱
        ╱╲
       ╱  ╲
      ╱    ╲
     ╱      ╲
    ╱        ╲

带宽收窄 → 波动率降低 → 即将变盘
```

**操作**:
- 收口末期减少操作
- 等待方向选择
- 突破后跟进

---

#### 2. 开口 (趋势确立)

```
    ╲            ╱
     ╲          ╱
      ╲        ╱
       ╲      ╱
        ╲    ╱
         ╲  ╱
          ╱╲
         ╱  ╲
        ╱    ╲
       ╱      ╲
      ╱        ╲

带宽扩大 → 波动率增加 → 趋势确立
```

**操作**:
- 向上开口：买入持股
- 向下开口：卖出持币

---

### 三、%B 指标

**公式**: 
```
%B = (收盘价 - 下轨) / (上轨 - 下轨)
```

**判断**:

| %B 值 | 位置 | 操作 |
|-------|------|------|
| > 1 | 上轨上方 | 超买，警惕 |
| 0.8-1 | 接近上轨 | 强势，持股 |
| 0.5 | 中轨 | 中性，观望 |
| 0-0.2 | 接近下轨 | 超卖，关注 |
| < 0 | 下轨下方 | 严重超卖，反弹 |

## T05: 成交量分析

### 一、量价关系

#### 1. 正常量价

| 价格 | 成交量 | 含义 | 操作 |
|------|--------|------|------|
| ↗ | 放大 | 健康上涨 | 持股 |
| ↗ | 缩小 | 上涨乏力 | 警惕 |
| ↘ | 放大 | 恐慌下跌 | 卖出 |
| ↘ | 缩小 | 下跌减缓 | 观望 |
| → | 缩小 | 盘整 | 等待 |

---

#### 2. 异常量价

```
放量滞涨:
价格：→ (横盘)
成交量：██████ (放大)
        ↑
    主力出货 → 卖出

缩量新高:
价格：↗ (新高)
成交量：█ (缩小)
        ↑
    上涨乏力 → 警惕

地量地价:
价格：↘↓↓ (连续下跌)
成交量： (极度缩小)
        ↑
    底部信号 → 关注
```

---

### 二、量比分析

**公式**:
```
量比 = 当前成交量 / 过去 5 日平均成交量
```

**判断**:

| 量比 | 含义 | 操作 |
|------|------|------|
| > 3 | 放量 | 关注方向 |
| 1.5-3 | 温和放量 | 顺势 |
| 0.8-1.5 | 正常 | 观望 |
| < 0.8 | 缩量 | 等待 |

---

### 三、换手率分析

**公式**:
```
换手率 = 成交量 / 流通股本 × 100%
```

**判断**:

| 换手率 | 含义 | 操作 |
|--------|------|------|
| > 20% | 极度活跃 | 警惕 |
| 10-20% | 高度活跃 | 短线 |
| 5-10% | 活跃 | 正常 |
| 3-5% | 一般 | 观望 |
| < 3% | 低迷 | 等待 |

## T06: 波浪理论

### 一、艾略特波浪基础

**基本结构**:

```
驱动浪 (5 浪):
    ②   ④
   ↗ ↘ ↗ ↘ ⑤
  ↗   ③   ↗
 ↗①       ↗
          ↗

调整浪 (3 浪):
    A   C
   ↘ ↗ ↘
    ↘ B ↘
```

**完整周期**: 5 浪上涨 + 3 浪下跌 = 8 浪

---

### 二、波浪特性

#### 驱动浪特性

| 浪 | 特性 | 成交量 |
|----|------|--------|
| ① | 启动浪，不易识别 | 温和放大 |
| ② | 回调浪，不破①起点 | 缩小 |
| ③ | 主升浪，最长最强 | 明显放大 |
| ④ | 整理浪，复杂 | 缩小 |
| ⑤ | 末升浪，背离 | 萎缩 |

---

#### 调整浪特性

| 浪 | 特性 | 形态 |
|----|------|------|
| A | 下跌开始 | 简单/复杂 |
| B | 反弹浪 | 诱多 |
| C | 主跌浪 | 最长 |

---

### 三、斐波那契回调

**常用比例**: 0.382、0.5、0.618

```
上涨回调:
    ①起点
   ↗
  ↗  0.382  ← 浅回调
 ↗   0.5    ← 中回调
↗    0.618  ← 深回调 (常见)
②终点

操作:
- 0.618 位置是常见买入点
- 结合其他指标确认
```

---

### 四、波浪实战

#### 数浪原则

```
1. 浪②不破浪①起点
2. 浪③不是最短的
3. 浪④不进入浪①区域
4. 交替原则：浪②简单则浪④复杂
```

#### 操作策略

```
浪①: 观望 (不易识别)
浪②: 准备 (回调到位)
浪③: 买入 (主升浪)
浪④: 持股 (整理)
浪⑤: 卖出 (末升浪)
浪 A: 空仓 (下跌开始)
浪 B: 观望 (反弹)
浪 C: 空仓 (主跌浪)
```

## 技术指标综合应用

### 一、指标组合策略

#### 趋势 + 震荡组合

```
趋势指标 (MACD/均线): 定方向
震荡指标 (KDJ/RSI): 找时机

买入条件:
- MACD 金叉 (趋势向上)
- KDJ 低位金叉 (时机)
- 成交量放大 (确认)

卖出条件:
- MACD 死叉 (趋势向下)
- KDJ 高位死叉 (时机)
- 成交量放大 (确认)
```

---

#### 多指标共振

```
强烈买入信号:
✓ 均线多头排列
✓ MACD 金叉
✓ KDJ 金叉
✓ RSI > 50
✓ 成交量放大
✓ 布林带开口

6 个指标同时确认 → 高胜率
```

---

### 二、指标优先级

| 场景 | 首选指标 | 辅助指标 |
