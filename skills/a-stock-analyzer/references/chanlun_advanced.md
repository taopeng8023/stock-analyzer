## C01: 缠论中枢扩展

### 一、中枢延伸

**定义**: 中枢形成后，后续走势继续在中枢区间内震荡

```
中枢延伸示例：
    
    ┌────┐ ZG
    │ ①  │
    └────┘
    ┌────┐
    │ ②  │  ← 延伸
    └────┘
    ┌────┐
    │ ③  │  ← 延伸
    └────┘
    ┌────┘ ZD
```

**特征**:
- 至少 3 笔重叠
- 后续笔不突破 ZG/ZD
- 延伸次数越多，中枢越稳固

**操作意义**:
- 中枢内高抛低吸
- 等待方向选择
- 延伸 9 段以上可能升级

---

### 二、中枢扩张

**定义**: 两个同级别中枢的波动区间产生重叠

```
中枢扩张示例：

    ┌────┐ 中枢 1
    │ ZG1│
    └────┘ ZD1
         ┌────┐ 中枢 2
         │ ZG2│
         └────┘ ZD2
    
    重叠区间 = [max(ZD1,ZD2), min(ZG1,ZG2)]
```

**条件**:
- 两个中枢同级别
- 波动区间有重叠
- 重叠部分形成新中枢

**升级规则**:
- 9 段同级别中枢延伸 → 级别升级
- 2 个中枢扩张 → 可能形成更大级别中枢

---

### 三、中枢新生

**定义**: 原中枢被突破后，形成新的中枢

```
中枢新生示例：

    ┌────┐ 原中枢
    │    │
    └────┘
         ↗ 突破
         ┌────┐ 新生中枢
         │    │
         └────┘
```

**特征**:
- 原中枢被有效突破
- 新中枢位置更高 (上涨) 或更低 (下跌)
- 趋势延续信号

## C02: 缠论背驰详解

### 一、背驰类型

#### 1. 盘整背驰

**定义**: 中枢内的背驰，力度较弱

```
盘整背驰示例：

    ↘ a 段 (下跌)
    ┌────┐
    │中枢│
    └────┘
    ↘ b 段 (下跌)
    
    条件：
    - b 段创新低
    - MACD 绿柱面积 < a 段
    - 背驰强度一般
```

**操作**:
- 反弹力度有限
- 小仓位参与
- 快进快出

---

#### 2. 趋势背驰

**定义**: 两个中枢之间的背驰，力度强

```
趋势背驰示例 (下跌趋势):

    ↘ a 段
    ┌────┐ 中枢 A
    └────┘
    ↘ b 段
    ┌────┐ 中枢 B
    └────┘
    ↘ c 段 ← 背驰段
    
    条件：
    - c 段创新低
    - c 段 MACD < b 段
    - 强烈反转信号
```

**操作**:
- 一买确认
- 大仓位参与
- 持有周期长

---

### 二、背驰判断方法

#### 1. MACD 面积法

```python
# 背驰强度计算
def beichi_strength(price_low, macd_area_prev, macd_area_curr):
    """
    计算背驰强度
    """
    if macd_area_prev == 0:
        return 0
    
    # 背驰强度 = 1 - (当前面积 / 前次面积)
    strength = 1 - (macd_area_curr / macd_area_prev)
    
    return {
        'strength': strength,
        'level': '强背驰' if strength > 0.5 else '弱背驰',
        'action': '买入' if strength > 0.3 else '观望'
    }
```

**判断标准**:
- 强度 > 0.5: 强背驰，立即行动
- 强度 0.3-0.5: 中等背驰，可参与
- 强度 < 0.3: 弱背驰，观望

---

#### 2. 黄白线高度法

**DIF/DEA 高度比较**:

| 情况 | 价格 | DIF | 结论 |
|------|------|-----|------|
| 正常 | 新低 | 新低 | 无背驰 |
| 背驰 | 新低 | 不新低 | 标准背驰 |
| 强背驰 | 新低 | 上穿零轴 | 强烈背驰 |

---

#### 3. 红绿柱比较法

```
下跌背驰：

a 段绿柱：████████ (面积大)
c 段绿柱：████ (面积小)
          ↑
      背驰确认
```

**要点**:
- 比较相邻两段
- 看绿柱总面积
- 结合价格创新低

---

### 三、小转大

**定义**: 小级别背驰引发大级别反转

```
小转大示例：

日线：↘↘↘↘ (无明显背驰)
30 分钟：↘↘↘ (背驰明显)
5 分钟：↘ (背驰强烈)
        ↑
    小级别引发大反转
```

**特征**:
- 大级别无明显背驰
- 小级别背驰强烈
- 突然反转

**操作**:
- 多级别联立观察
- 小级别信号先行
- 快速反应

## C03: 缠论区间套

### 一、多级别联立

**原理**: 大级别定方向，小级别找买卖点

```
级别联立示例：

日线：    ↗↗↗ (上涨趋势)
          ↑ 方向

30 分钟：  ↗↘↗ (回调结束)
          ↑ 时机

5 分钟：   ↗突破 (买入信号)
          ↑ 精确点位
```

**操作流程**:
1. 日线定方向 (上涨/下跌/盘整)
2. 30 分钟找位置 (中枢/突破)
3. 5 分钟抓买点 (精确入场)

---

### 二、级别对应关系

| 大级别 | 中级别 | 小级别 | 用途 |
|--------|--------|--------|------|
| 月线 | 周线 | 日线 | 长线投资 |
| 周线 | 日线 | 60 分钟 | 中线波段 |
| 日线 | 30 分钟 | 5 分钟 | 短线操作 |
| 30 分钟 | 5 分钟 | 1 分钟 | 超短线 |

---

### 三、精确定位买卖点

#### 一买定位 (5 分钟级别)

```
日线：    ↘↘↘ (下跌趋势)
          ↓ 等待

30 分钟：  ↘↘ 底背驰
          ↓ 准备

5 分钟：   ↘↗ 突破
          ↑ 买入！
```

**步骤**:
1. 日线确认下跌趋势
2. 30 分钟出现底背驰
3. 5 分钟突破下降趋势线
4. 精确入场点

---

#### 三买定位 (多级别确认)

```
日线：    突破中枢 ↗
          ↓ 观察

30 分钟： 回踩不破 ↗
          ↓ 确认

5 分钟：   再次启动 ↗
          ↑ 买入！
```

## C04: 缠论走势分解

### 一、走势类型连接

**三种走势类型**:
1. **上涨**: 高点抬高，低点抬高
2. **下跌**: 高点降低，低点降低
3. **盘整**: 高低点交错

**连接规则**:
```
上涨 + 下跌 = 转折
下跌 + 上涨 = 转折
上涨 + 上涨 = 延续
下跌 + 下跌 = 延续
盘整 + 任何 = 等待突破
```

---

### 二、同级别分解

**原则**: 将走势按同一级别分解

```
示例 (30 分钟级别分解):

    a↘  b↗  c↘  d↗  e↘
    └──┘ └──┘ └──┘
     ①    ②    ③   ← 中枢
    
分解为：
- 段 a: 进入段
- 中枢①②③: 盘整
- 段 e: 离开段
```

**优势**:
- 结构清晰
- 便于判断背驰
- 买卖点明确

---

### 三、走势多义性

**定义**: 同一走势可能有多种分解方式

```
示例：

分解 1: a↘ + 中枢 + b↗
分解 2: 大级别上涨中的一段

处理原则：
- 选择最简洁的分解
- 符合结合律
- 便于操作
```

## C05: 缠论动力学

### 一、力度比较

**比较对象**:
1. 进入段 vs 离开段
2. 相邻两段
3. 中枢前后

```
力度比较示例：

a 段：长度 10，MACD 面积 100
b 段：长度 12，MACD 面积 80
          ↑
      力度减弱 → 背驰
```

**判断**:
- 价格创新高/新低
- MACD 面积减小
- 力度背驰确认

---

### 二、黄白线动力学

**DIF/DEA 运行规律**:

```
零轴上方：
- DIF/DEA > 0: 多头市场
- 回踩零轴不破：二买

零轴下方：
- DIF/DEA < 0: 空头市场
- 反弹零轴不过：二卖

穿越零轴：
- 下穿零轴：转空
- 上穿零轴：转多
```

---

### 三、红绿柱动力学

**红绿柱变化规律**:

```
上涨趋势：
绿柱缩短 → 红柱出现 → 红柱变长 → 红柱缩短 → 绿柱出现
  ↓          ↓          ↓          ↓          ↓
止跌      转多      强势      减弱      转空
```

**操作要点**:
- 绿转红：买入信号
- 红转绿：卖出信号
- 红柱最长：持股
- 绿柱最长：持币
