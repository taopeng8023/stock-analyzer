## K01: 蜡烛图组合策略

### 一、多形态确认

**原理**: 单一形态可靠性有限，多个形态确认提高胜率

#### 1. 底部多重确认

```
底部组合示例：

    ↘ 大阴线
    ┌─┴─┐
    │   │
    └─┬─┘
      │  锤头线 ✓
    ──┼──
      │  十字星 ✓
    ┌─┴─┐
    │大阳│  看涨吞没 ✓
    └───┘
    
    三个形态确认 → 强烈买入信号
```

**组合类型**:
- 锤头线 + 十字星 + 看涨吞没 = 强烈底部
- 早晨之星 + 看涨吞没 = 高胜率底部
- 刺透形态 + 红三兵 = 趋势反转

---

#### 2. 顶部多重确认

```
顶部组合示例：

      ┌─┴─┐
      │大阳│
      └─┬─┘
        │  射击之星 ✓
      ──┼──
        │  十字星 ✓
    ┌─┴─┐
    │大阴│  看跌吞没 ✓
    └───┘
    
    三个形态确认 → 强烈卖出信号
```

---

### 二、位置判断

**同一形态，不同位置，含义不同**

#### 1. 关键位置的锤头线

```
支撑位锤头线 (高胜率):

阻力位 ────────
        ↘
        ↘
支撑位 ──┬─  锤头线
       ┌─┴─┐
       │阳│
       └───┘
       ↑
   强支撑 + 锤头线 = 高胜率买入

中间位置锤头线 (低胜率):

        ┌─┴─┐
        │阳│  锤头线
        └───┘
        ↑
    中间位置 = 普通信号
```

**关键位置**:
- 前高/前低
- 中枢上沿/下沿
- 均线支撑/压力
- 斐波那契回调位 (0.618)
- 整数关口

---

#### 2. 突破位置的吞没形态

```
突破中枢的看涨吞没 (高胜率):

    ┌────┐ ZG
    │中枢│
    └────┘
    ┌─┴─┐
    │大阳│  吞没阴线 + 突破中枢
    └───┘
    ↑
突破 + 吞没 = 强烈买入

中枢内的看涨吞没 (低胜率):

    ┌────┐
    │吞没│  中枢内震荡
    └────┘
    ↑
中枢内 = 普通信号
```

---

### 三、形态强度排序

| 强度 | 形态组合 | 胜率 | 操作 |
|------|---------|------|------|
| ⭐⭐⭐⭐⭐ | 早晨之星 + 吞没 + 突破 | >80% | 重仓 |
| ⭐⭐⭐⭐ | 黄昏之星 + 吞没 | 70-80% | 中仓 |
| ⭐⭐⭐ | 单一吞没形态 | 60-70% | 轻仓 |
| ⭐⭐ | 单一锤头/射击之星 | 50-60% | 观望 |

## K02: 蜡烛图 + 均线

### 一、均线过滤假信号

**问题**: 蜡烛图在震荡市中假信号多

**解决**: 用均线过滤，只在趋势方向操作

#### 1. 多头均线 + 看涨蜡烛

```python
# 买入条件
if MA5 > MA10 > MA20:  # 多头排列
    if 出现看涨吞没 or 早晨之星:
        if 股价 > MA20:  # 在均线上方
            return "买入信号 (高胜率)"
        else:
            return "观望 (跌破均线)"
else:
    return "观望 (非多头趋势)"
```

**示例**:

```
MA5 ╱
MA10 ╱   ← 多头排列
MA20 ╱

    ┌─┴─┐
    │大阳│  看涨吞没
    └───┘
    ↑
均线上方 + 看涨形态 = 买入
```

---

#### 2. 空头均线 + 看跌蜡烛

```python
# 卖出条件
if MA5 < MA10 < MA20:  # 空头排列
    if 出现看跌吞没 or 黄昏之星:
        if 股价 < MA20:  # 在均线下方
            return "卖出信号 (高胜率)"
        else:
            return "观望 (突破均线)"
else:
    return "观望 (非空头趋势)"
```

---

### 二、均线支撑/压力 + 蜡烛确认

#### 1. 均线支撑买入

```
MA20 支撑 + 锤头线:

MA5 ╱
MA10 ╱
MA20 ╱════════  ← 支撑位
         ┌─┴─┐
         │阳│  锤头线
         └───┘
         ↑
    支撑确认 → 买入
```

**操作**:
- 股价回调至 MA20
- 出现锤头线/十字星
- 次日阳线确认
- 买入，止损 MA20 下方

---

#### 2. 均线压力卖出

```
MA20 压力 + 射击之星:

         ┌─┴─┐
         │阴│  射击之星
         └───┘
MA20 ╲════════  ← 压力位
MA10 ╲
MA5 ╲
         ↑
    压力确认 → 卖出
```

**操作**:
- 股价反弹至 MA20
- 出现射击之星/黄昏之星
- 次日阴线确认
- 卖出，止损 MA20 上方

---

### 三、均线周期配合

| 蜡烛形态 | 最佳均线周期 | 适用场景 |
|----------|-------------|---------|
| 锤头线/上吊线 | MA20 | 短线回调/反弹 |
| 吞没形态 | MA20/MA60 | 趋势反转 |
| 早晨/黄昏之星 | MA60 | 中线趋势反转 |
| 红三兵/三只乌鸦 | MA20/MA60 | 趋势确认 |

## K03: 蜡烛图 + 缠论

### 一、分型确认

**缠论分型 + 蜡烛形态 = 高胜率信号**

#### 1. 顶分型确认

```
顶分型 + 射击之星 (强烈卖出):

      ┌─┴─┐  ← 中间 K 线 (最高点)
      │   │
    ┌─┴─┐ │
    │阳│ │
    └───┘ │
          └─┬─┘
            │  射击之星
          ┌─┴─┐
          │阴│
          └───┘
          ↑
    顶分型 + 射击之星 = 一卖确认
```

**操作**:
- 识别顶分型
- 中间 K 线是射击之星/黄昏之星
- 第三根 K 线阴线确认
- 卖出

---

#### 2. 底分型确认

```
底分型 + 锤头线 (强烈买入):

          └─┬─┘
            │  锤头线
          ┌─┴─┐
          │阳│
          └───┘
    ┌─┴─┐ │
    │阴│ │
    └───┘ │
      ┌─┴─┐  ← 中间 K 线 (最低点)
      │   │
      ↑
底分型 + 锤头线 = 一买确认
```

**操作**:
- 识别底分型
- 中间 K 线是锤头线/早晨之星
- 第三根 K 线阳线确认
- 买入

### 二、仓位管理

```python
def 仓位计算 (蜡烛信号，均线，缠论，背驰):
    基础仓位 = 20%
    
    # 蜡烛形态加分
    if 三重确认:
        基础仓位 += 40%
    elif 双重确认:
        基础仓位 += 30%
    elif 单一形态:
        基础仓位 += 10%
    
    # 均线加分
    if 均线多头 and 看涨:
        基础仓位 += 20%
    elif 均线空头 and 看涨:
        基础仓位 -= 10%
    
    # 缠论加分
    if 缠论一买:
        基础仓位 += 20%
    elif 缠论三买:
        基础仓位 += 30%
    
    # 背驰加分
    if 背驰强度 > 0.5:
        基础仓位 += 20%
    
    return min(基础仓位，100%)
```
