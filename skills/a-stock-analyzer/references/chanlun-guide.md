# 缠论快速参考指南

## 缠论基础

缠论 (Chan Theory) 是由"缠中说禅"创建的技术分析理论体系，核心思想是**走势终完美**，即任何走势最终都会完成。

## 核心概念速查

### 1. 分型 (Fractal)

```
顶分型：    底分型：
   /\          \/
  /  \        /  \
 /    \      /    \
高 - 更高 - 高   低 - 更低 - 低
```

**识别规则**:
- 顶分型：中间 K 线高点最高，低点也最高
- 底分型：中间 K 线低点最低，高点也最低
- 相邻分型至少间隔 4 根 K 线

**操作意义**:
- 顶分型 → 潜在卖出点
- 底分型 → 潜在买入点

---

### 2. 笔 (Bi)

```
向上笔：      向下笔：
  /\            \/
 /  \          /  \
/    \        /    \
底→顶        顶→底
```

**形成条件**:
1. 顶底分型交替出现
2. 顶分型高点 > 底分型低点 (向上笔)
3. 至少 5 根 K 线 (包含处理后)

**操作意义**:
- 向上笔延续 → 持股
- 向下笔延续 → 持币
- 笔方向改变 → 警惕趋势反转

---

### 3. 线段 (Duan)

```
上涨线段：至少 3 笔向上
  /\  /\  /\
 /  \/  \/  \

下跌线段：至少 3 笔向下
  \  /\  /\  /
   \/  \/  \/
```

**形成条件**:
- 至少 3 笔
- 方向一致
- 不被反向突破

**操作意义**:
- 线段是操作的基本单位
- 线段破坏 = 操作信号

---

### 4. 中枢 (Zhongshu)

```
     ┌────────┐ ZG (中枢上沿)
     │  重叠  │
─────┼────────┼─────
     │  区间  │
     └────────┘ ZD (中枢下沿)
```

**定义**: 至少 3 笔重叠的价格区间

**关键位置**:
- **ZG**: 中枢上沿 = min(高点)
- **ZD**: 中枢下沿 = max(低点)
- **ZZ**: 中枢中点 = (ZG + ZD) / 2

**操作意义**:
- 中枢内 → 震荡操作
- 突破中枢 → 趋势行情
- 回踩中枢 → 确认信号

---

### 5. 背驰 (Beichi)

```
底背驰：
价格：新低 ────↘
MACD: 不新低 ──→  ← 背离

顶背驰：
价格：新高 ────↗
MACD: 不新高 ──→  ← 背离
```

**识别方法**:
1. 比较相邻同向笔
2. 价格创新高/新低
3. MACD 面积不创新高/新低

**背驰强度**:
- 强度 = 1 - (后笔 MACD 面积 / 前笔 MACD 面积)
- 强度 > 0.5 → 强背驰
- 强度 > 0.7 → 极强背驰

---

### 6. 三类买卖点

#### 第一类买卖点 (趋势反转)

```
一买：          一卖：
    \              /
     \ 底背驰     / 顶背驰
      \  ↓        /  ↓
       \        /
        ↘      ↙
```

**特征**:
- 一买：下跌趋势 + 底背驰
- 一卖：上涨趋势 + 顶背驰

**风险**: 最高 (可能抄在半山腰)
**收益**: 最大 (抓到趋势反转)

---

#### 第二类买卖点 (回调确认)

```
二买：          二卖：
    \              /
     \ 一买       / 一卖
      \  ↗       /  ↘
       \ /      \ /
        ↓        ↓
      不破低    不创新高
```

**特征**:
- 二买：一买后回调，不破前低
- 二卖：一卖后反弹，不破前高

**风险**: 中等
**收益**: 中高

---

#### 第三类买卖点 (突破确认)

```
三买：                三卖：
     ┌────┐               ┌────┐
     │中枢│               │中枢│
     └────┘               └────┘
        ↑                    ↓
     突破回踩              跌破反弹
        ↓                    ↑
      不破 ZG              不过 ZD
```

**特征**:
- 三买：突破中枢后回踩，不破 ZG
- 三卖：跌破中枢后反弹，不过 ZD

**风险**: 最低 (信号最确认)
**收益**: 中等

## 实战操作流程

### 步骤 1: 画分型
1. 识别顶分型和底分型
2. 过滤无效分型 (距离太近)

### 步骤 2: 连笔
1. 连接相邻顶底分型
2. 验证笔的有效性 (≥5 根 K 线)

### 步骤 3: 成线段
1. 至少 3 笔同向
2. 确认线段未被破坏

### 步骤 4: 找中枢
1. 寻找 3 笔重叠区间
2. 标记 ZG 和 ZD

### 步骤 5: 判背驰
1. 比较相邻同向笔
2. 计算 MACD 面积比

### 步骤 6: 定买卖点
1. 识别一/二/三买卖点
2. 结合仓位管理操作

## 常见问题

### Q1: 为什么有时候画不出笔？
A: 分型之间 K 线数量不足 5 根，或价格不满足条件。

### Q2: 中枢一定水平吗？
A: 不一定。中枢可以倾斜，关键是 3 笔有重叠区间。

### Q3: 背驰就一定反转吗？
A: 不一定。背驰只是必要条件，还需其他信号确认。

### Q4: 如何区分盘整背驰和趋势背驰？
A: 
- 盘整背驰：中枢内的背驰，力度较弱
- 趋势背驰：两个中枢之间的背驰，力度较强

### Q5: 多级别联立怎么用？
A: 大级别定方向，小级别找买卖点。例如：
- 日线向上 → 30 分钟找二买/三买
- 日线向下 → 30 分钟找二卖/三卖

## 风险提示

1. **缠论是概率游戏** - 没有 100% 准确的信号
2. **级别很重要** - 不同级别信号可能矛盾
3. **结合基本面** - 纯技术分析有局限
4. **仓位管理** - 再好的信号也要控制仓位
5. **止损纪律** - 信号失败要果断止损
