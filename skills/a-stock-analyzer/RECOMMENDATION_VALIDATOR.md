# 推荐验证器 - 股票推荐质量保证系统

> **创建日期**: 2026-04-06  
> **触发事件**: 错误推荐中闽能源 (600163)，事后回测显示胜率仅 28.57%

---

## 📋 核心原则：三不推荐

1. **不回测不推荐** — 任何股票必须先跑回测，胜率<70% 不推荐
2. **不说明风险不推荐** — 必须告知历史胜率和最大回撤
3. **不跟踪不推荐** — 推荐后需跟踪表现，计入错误日志

---

## ✅ 推荐检查清单

推荐股票前必须验证：

```
□ 历史回测胜率 ≥ 70%
□ 盈亏比 ≥ 2.0
□ 最大回撤 < 25%
□ 年交易次数 ≥ 3 次（避免偶发性）
□ 基本面没有重大利空
□ 不属于问题股黑名单
```

---

## 🎯 推荐等级

| 等级 | 胜率要求 | 盈亏比要求 | 说明 |
|------|---------|-----------|------|
| 🌟 **强烈买入** | ≥80% | ≥5.0 | 各项指标优秀，可重点配置 |
| ✅ **买入** | ≥70% | ≥2.0 | 符合标准，可正常配置 |
| 📊 **持有** | ≥60% | ≥1.5 | 表现良好，继续观察 |
| 👀 **观察** | ≥50% | ≥1.0 | 符合条件但非最优 |
| 📉 **卖出** | <50% | <1.0 | 未达推荐标准 |
| 🚫 **避免** | <40% | - | 存在严重问题，建议回避 |

## 🔧 使用方法

### 1. 单独使用验证器

```python
from recommendation_validator import validate_stock

# 验证中闽能源 (错误案例)
result = validate_stock(
    stock_code='600163',
    win_rate=28.57,      # 胜率
    profit_loss_ratio=0.10,  # 盈亏比
    annual_return=6.59,  # 年化收益
    max_drawdown=20.39,  # 最大回撤
    total_trades=7,      # 交易次数
    sharpe_ratio=0.14    # 夏普比率
)

print(result.passed)   # False
print(result.level)    # RecommendationLevel.AVOID
print(result.issues)   # ['胜率过低：28.57% (要求≥70.0%)', ...]
```

### 2. 集成到分析脚本

```bash
# 分析股票（自动包含验证）
python scripts/analyze_stock.py 002709

# JSON 输出
python scripts/analyze_stock.py 002709 --json
```

### 3. 黑名单管理

```python
from recommendation_validator import RecommendationValidator

validator = RecommendationValidator()

# 添加股票到黑名单
validator.add_to_blacklist('600163', '胜率过低 28.57%')

# 黑名单自动保存到 data/blacklist.json
```

## 📊 验证指标说明

| 指标 | 阈值 | 说明 |
|------|------|------|
| 胜率 | ≥70% | 历史交易盈利概率 |
| 盈亏比 | ≥2.0 | 平均盈利/平均亏损 |
| 年化收益 | ≥15% | 年化收益率 |
| 最大回撤 | <25% | 历史最大亏损幅度 |
| 交易次数 | ≥3 次/年 | 避免偶发性信号 |
| 夏普比率 | ≥0.5 | 风险调整后收益 |
