### B. 综合评分排行

**评分公式**：
```
综合评分 = 成交额得分 × 40% + 涨跌幅得分 × 40% + 成交量得分 × 20%
```

**计算方法**：
```python
# 成交额得分（真实成交额）
amount_score = min(100, amount_wan / 100000 * 10)

# 涨跌幅得分（真实涨跌幅）
change_score = min(100, max(0, change_pct + 50))

# 成交量得分（真实成交量）
volume_score = min(100, volume / 10000000 * 10)

# 综合评分
score = amount_score * 0.4 + change_score * 0.4 + volume_score * 0.2
```
