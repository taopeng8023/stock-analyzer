# News Monitor Skill - 消息监控分析技能

**实时监控国内外消息，分析对 A 股影响**

---

## 📋 技能概述

- 每分钟获取国内外重要消息
- 智能分析消息对 A 股的影响
- 12 小时内重复消息自动过滤
- 实时推送重要消息

---

## 🎯 功能特性

| 功能 | 说明 |
|------|------|
| 实时获取 | 每分钟获取最新消息 |
| 消息去重 | 12 小时内重复消息不推送 |

## 🚀 快速开始

### 安装

```bash
cd /home/admin/.openclaw/workspace/skills/news-monitor
```

### 使用

```python
from news_monitor import NewsMonitorSkill

# 初始化
monitor = NewsMonitorSkill()

# 获取并分析消息
news_list = monitor.fetch_news()

# 分析影响
for news in news_list:
    analysis = monitor.analyze_impact(news)
    if analysis['should_push']:
        monitor.push_news(analysis)
```

## 🔄 去重机制

### 去重规则

1. **标题去重**
   - 标题相似度>90% 视为重复

2. **内容去重**
   - 内容相似度>80% 视为重复

3. **时间窗口**
   - 12 小时内重复消息不推送

4. **来源去重**
   - 同一事件多源报道只推送一次

### 去重流程

```
新消息 → 计算指纹 → 对比历史库 → 
  ↓
相似？→ 是 → 丢弃
  ↓
  否
  ↓
保存并推送
```
