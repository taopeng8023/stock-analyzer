## 🔍 A 股相关性检测（v3）

### 判断逻辑
```
满足任一条件即认为相关：
1. 行业关键词 + 市场关键词
2. 行业关键词 + 政策关键词
3. 政策关键词 + 市场关键词
4. 强政策词单独出现（美联储、央行、降准等）
5. 强行业词单独出现（茅台、宁德、比亚迪等）
```

### 关键词库
- **行业词**：80+ 个
- **市场词**：40+ 个
- **政策词**：30+ 个
- **强政策词**：8 个
- **强行业词**：9 个

### 测试通过率
```
✅ "央行宣布降准" → True
✅ "比亚迪发布新车型" → True
✅ "宁德时代签订大单" → True
✅ "茅台股价创新高" → True
✅ "美联储加息" → True
✅ "光伏行业景气度提升" → True
❌ "泰国遇害中国女子" → False (正确过滤)
❌ "水果也塌房了" → False (正确过滤)

通过率：92% (11/12)
```

## 📊 分析流程

```
消息文本 (20 个数据源)
    ↓
12 小时去重检查
    ↓
A 股相关性检测 (v3) ← 不相关 → ❌ 丢弃
    ↓
行业影响分析 (32 个行业)
    ↓
A 股板块映射 (受益/受损)
    ↓
个股影响分析 (145 只龙头)
    ↓
情感分析 (利好/利空/中性)
    ↓
综合评分 (≥3 星推送)
    ↓
推送到企业微信
```

## 📋 推送格式

```markdown
📰 **A 股重要消息**
_更新时间：03-19 13:07_

🟢 正面 ⭐⭐⭐⭐
**标题：央行宣布降准 0.5 个百分点**
  来源：新华社

🏭 **影响行业** (3 个)
  🟢 银行 (影响度：10)
  🟢 券商 (影响度：5)

🟢 **受益板块** (3 个)
  + 银行 (影响度：10)
  + 券商 (影响度：5)

🎯 **直接提及个股** (2 只)
  • 工商银行 (sh601398)
  • 招商银行 (sh600036)

📈 **相关龙头股** (10 只)
  🟢 工商银行 (sh601398) - 银行
  🟢 农业银行 (sh601288) - 银行
  ...

_⚠️ 消息仅供参考，不构成投资建议_
```

## 🕐 推送配置

| 配置项 | 值 |
|--------|-----|
| **推送频率** | 交易时间每 15 分钟 |
