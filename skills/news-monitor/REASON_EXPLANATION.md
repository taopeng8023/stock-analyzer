# 个股影响原因说明功能 - v6.1

**更新时间**：2026-03-19 18:00
**版本**：v6.1

---

## 📋 功能说明

在消息分析推送时，为每个关联的个股添加**原因说明**，让用户了解为什么这条消息会影响这只股票。

---

## 📊 推送格式对比

### v6.0（无原因）
```
📈 **相关龙头股** (6 只)
  🟢 工商银行 (sh601398) - 银行
     📈 现价：¥7.53 | 涨跌：+1.89% | 成交：420252.0 亿
```

### v6.1（含原因）⭐
```
📈 **相关龙头股** (6 只)
  🟢 工商银行 (sh601398) - 银行
     📈 现价：¥7.53 | 涨跌：+1.89% | 成交：420252.0 亿
     💡 央行降准/降息政策直接利好银行板块，提升信贷投放能力（关键词：银行、降准）
```

---

## 🔍 原因生成逻辑

### 1. 板块匹配
根据消息内容匹配到受影响的板块（如银行、AI、半导体等）。

### 2. 关键词提取
从消息中提取与板块相关的关键词。

### 3. 原因模板
每个板块+情感组合有预设的原因说明模板。

### 4. 动态生成
将匹配的关键词填入模板，生成最终原因。

## 🔧 自定义原因模板

可以在 `stock_analyzer.py` 的 `reason_templates` 字典中添加新的原因模板：

```python
reason_templates = {
    ('板块名', 'positive'): '正面原因模板',
    ('板块名', 'negative'): '负面原因模板',
    ...
}
```
