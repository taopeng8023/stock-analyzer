## 🔧 API 说明

### 今日头条
- **URL**: `https://www.toutiao.com/api/pc/feed/?category=news_hot`
- **类型**: 热门推荐
- **字段**: title, abstract, behot_time, item_id, url
- **限制**: 无需 API key，有反爬

### 知乎热榜
- **URL**: `https://www.zhihu.com/api/v3/feed/topstory/hot?limit=20`
- **类型**: 热榜话题
- **字段**: title, headline, excerpt, url
- **限制**: 无需 API key，有反爬

### 华尔街见闻
- **URL**: `https://api.wallstreetcn.com/apiv1/content/lives?channel=global`
- **类型**: 全球财经快讯
- **字段**: title, content, published_at, id
- **限制**: 无需 API key

### 金十数据
- **URL**: `https://www.jin10.com/flash_newest.js`
- **类型**: 财经数据快讯
- **字段**: content, time, id
- **限制**: JSONP 格式，需解析

### 腾讯财经
- **URL**: `https://r.inews.qq.com/gw/event/more_hot_ranking_list`
- **类型**: 热门财经新闻
- **字段**: title, abstract, timestamp, url
- **限制**: 无需 API key

### 同花顺
- **URL**: `http://news.10jqka.com.cn/interface/news/list`
- **类型**: 财经新闻
- **字段**: title, content, time, url
- **限制**: JSONP 格式

### 新浪财经
- **URL**: `https://feed.mix.sina.com.cn/api/roll/get?pageid=153&lid=2509`
- **类型**: 财经新闻
- **字段**: title, intro, ctime, url
- **限制**: 有时返回 403

## ⚠️ 注意事项

1. **所有数据源均为真实市场数据** - 严格遵守"不使用模拟数据"原则
2. **API 可能变化** - 部分 API 无官方文档，可能随时变更
3. **反爬机制** - 部分源有反爬，需设置合理的 User-Agent
4. **去重机制** - 12 小时内重复消息自动过滤
5. **推送策略** - 只推送影响等级≥3 星的重要消息
