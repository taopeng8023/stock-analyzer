# 消息推送策略

**版本：v3.0**
**更新时间：2026-03-18**

---

## 🎯 核心原则

> **只推送可能影响 A 股的重要消息**

### 三条红线

1. ✅ **只推送与 A 股相关的消息** - 过滤社会、娱乐新闻
2. ✅ **只推送 3 星及以上消息** - 避免轻微影响消息打扰
3. ✅ **每小时整点推送一次** - 避免过度打扰

---

## 📊 推送流程

```
获取消息 (7 个数据源)
    ↓
去重检查 (12 小时窗口)
    ↓
A 股相关性检查 ← 不相关 → ❌ 丢弃
    ↓
影响分析评分
    ↓
评分≥3 星？ ← 否 → ❌ 不推送
    ↓ 是
✅ 推送到企业微信
```

## 🔍 A 股相关性判断

### 关键词库（50+ 个）

| 类别 | 关键词示例 |
|------|-----------|
| **市场相关** | a 股、沪深、上证、创业板、科创板、涨停、跌停 |
| **政策相关** | 证监会、央行、降准、降息、发改委 |
| **行业板块** | 银行、证券、芯片、新能源、医药、白酒、地产 |
| **公司相关** | 财报、业绩、利润、并购、重组、股东 |
| **资金相关** | 北向资金、外资、主力、资金流 |
| **宏观经济** | gdp、cpi、ppi、美联储、利率、汇率 |
| **重要指数** | 上证指数、沪深 300、创业板指 |

### 判断逻辑

```python
def _is_a_share_related(text):
    return any(keyword in text for keyword in a_share_keywords)
```

## ⭐ 影响评分系统

### 高权重关键词（每个 +2 分）

- 降准、降息、央行
- 证监会、财政部、发改委
- 涨停、跌停、突破、新高
- ipo、上市、重组、并购
- 财报、业绩、利润增长

### 普通关键词（每个 +1 分）

**利好**：支持、增长、扶持、上涨、突破、回暖、复苏

**利空**：限制、下降、风险、处罚、收紧、崩盘、恶化

### 评分等级

| 总分 | 星级 | 推送 |
|------|------|------|
| ≥5 分 | ⭐⭐⭐⭐⭐ | ✅ 推送 |
| 4 分 | ⭐⭐⭐⭐ | ✅ 推送 |
| 3 分 | ⭐⭐⭐ | ✅ 推送 |
| 2 分 | ⭐⭐ | ❌ 不推送 |
| 1 分 | ⭐ | ❌ 不推送 |
| 0 分 | - | ❌ 不相关 |

## ⚠️ 注意事项

1. **非交易时间** - 周末和节假日也会推送重要消息
2. **突发重大事件** - 可能在非整点时间推送
3. **消息质量优先** - 宁可少推，不可乱推
4. **持续优化** - 根据实际效果调整关键词和评分
