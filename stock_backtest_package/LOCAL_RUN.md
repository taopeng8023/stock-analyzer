```bash

```

## ⚠️ 注意事项

### 1. 网络要求

- 需要能访问东方财富 API
- 如果在中国大陆，通常可以直接访问
- 如果在海外，可能需要代理

### 2. 数据缓存

首次运行会下载数据，后续运行会自动使用缓存：
```
data/kline/600549.json  # K 线数据缓存
```

### 3. API 限流

东方财富 API 有限流，批量回测时建议添加延时：
```python
time.sleep(1)  # 每只股票间隔 1 秒
```

## 📞 技术支持

**问题反馈：**
- 查看 README.md
- 检查网络连接
- 查看错误日志

**常见问题：**

**Q: 无法获取数据？**
A: 检查网络连接，确保能访问东方财富网站

**Q: 回测无交易？**
A: 降低买入阈值（75 → 70）

**Q: 数据获取慢？**
A: API 限流，添加延时 `time.sleep(1)`
