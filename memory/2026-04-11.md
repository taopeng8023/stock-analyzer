### ⚠️ 重要修正：RSI计算方法错误

**发现时间：19:54**

**问题：**
主人发现皇台酒业在同花顺显示RSI=41.23，但我的脚本显示RSI=35.0

**原因分析：**
| 计算方法 | 皇台酒业RSI | 说明 |
|---------|------------|------|
| 简单平均法(我的旧方法) | 38.89 | ❌ 不标准 |
| SMA方法 | 38.98 | ❌ |
| **EMA/Wilders平滑法** | **41.23** | ✅ 同花顺标准 |

**结论：同花顺使用EMA/Wilders平滑法计算RSI**

**已修正：**
- `daily_stock_pick.py` 中的 `calc_rsi()` 函数
- 改用EMA平滑法(与同花顺一致)

**修正后结果：**
- 皇台酒业(000995) RSI: 41.23 ✅
- 符合条件股票: 1220只(正确筛选)
- 皇台酒业排在第340位(RSI较高的后排)

**代码修正要点：**
```python
# 旧方法(错误)：
gains = sum(涨幅) / 14
losses = sum(跌幅) / 14

# 新方法(正确) - Wilders平滑：
# 初始平均 + 后续平滑
gains = (gains*(n-1) + gain) / n
losses = (losses*(n-1) + loss) / n
```

**教训：**
- 计算指标必须与主流软件一致
- 发现差异立即核实
- 主人截图纠错很重要
