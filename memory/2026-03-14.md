### 重要说明

1. **验证码限制**: 东方财富网有滑块验证码，必须通过 browser 工具手动完成
2. **数据获取流程**:
   - `browser open https://data.eastmoney.com/zjlx/detail.html`
   - 手动完成验证码
   - 点击对应排行选项卡（今日/3 日/5 日/10 日）
   - 等待数据加载后滚动页面
   - 使用 evaluate 提取数据
   - 用 `process_data.py` 处理保存

3. **3 日排行数据格式**（根据实际页面）:
   - 3 日涨跌幅（累计）
   - 3 日主力净流入（净额 + 净占比）
   - 3 日超大单净流入（净额 + 净占比）
   - 3 日大单/中单/小单净流入
